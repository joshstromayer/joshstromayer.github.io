What Are Lists?

Lists are a way to organize content in HTML. They allow you to group related items together in a structured and readable way.


Types of Lists

In HTML, there are two main types of lists:

  • Ordered Lists (<ol>): A numbered list, where each item has a sequence number.
  • Unordered Lists (<ul>): A bulleted list, where the items have no specific order.

List Items

Both ordered and unordered lists use List Items (`<li>`) to define each individual item within the list.


Examples

Here’s an example of each type of list:

Ordered List:
<ol>
  <li>First Item</li>
  <li>Second Item</li>
  <li>Third Item</li>
</ol>

Unordered List:
<ul>
  <li>First Item</li>
  <li>Second Item</li>
  <li>Third Item</li>
</ul>

Unordered Lists

An unordered list displays a list of items without any specific order. Each item is typically marked with a bullet point.


To create an unordered list, use the <ul> tag. Inside the unordered list, use the <li> tag to define each item.


For example:

<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Oranges</li>
</ul>

This code will render the following list:

  • Apples
  • Bananas
  • Oranges

Key Points:

  • The <ul> tag defines the unordered list.
  • The <li> tag defines each individual item within the list.
  • Unordered lists are useful for displaying items where the order does not matter.

Ordered Lists

An ordered list displays a list of items in a specific order. Each item is typically marked with a number.


To create an ordered list, use the <ol> tag. Inside the ordered list, use the <li> tag to define each item.


For example:

<ol>
  <li>Preheat the oven</li>
  <li>Mix the ingredients</li>
  <li>Bake for 30 minutes</li>
</ol>

This code will render the following list:

  1. Preheat the oven
  2. Mix the ingredients
  3. Bake for 30 minutes

Key Points:

  • The <ol> tag defines the ordered list.
  • The <li> tag defines each individual item within the list.
  • Ordered lists are useful for steps, instructions, or any content where the order matters.

Nested Lists

Sometimes, you might want to create a list within another list. This is called a nested list.


You can nest either <ul> (unordered lists) or <ol> (ordered lists) inside each other—or even mix them!


For example:

<ul>
  <li>Fruits</li>
  <ul>
    <li>Apple</li>
    <li>Banana</li>
    <li>Orange</li>
  </ul>
  <li>Vegetables</li>
  <ul>
    <li>Carrot</li>
    <li>Broccoli</li>
    <li>Spinach</li>
  </ul>
</ul>

This code will render the following list:

  • Fruits
    • Apple
    • Banana
    • Orange
  • Vegetables
    • Carrot
    • Broccoli
    • Spinach

Key Points:

  • To create a nested list, place a new <ul> or <ol> inside an <li> tag.
  • Nested lists are useful for organizing hierarchical information.
