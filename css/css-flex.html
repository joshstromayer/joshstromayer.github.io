CSS Flex

The flex property in CSS (short for Flexbox) is used to create one-dimensional layouts where items are aligned and distributed efficiently along a row or column. Flexbox is ideal for creating responsive layouts and handling alignment and spacing.

Setting Up a Flex Container

To use Flexbox, set display: flex on the container element. The direct children of this container automatically become flex items.

<style>
.flex-container {
  display: flex;
  background-color: lightgray;
  padding: 10px;
}
.flex-item {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
  margin: 5px;
}
</style>
<div class="flex-container">
  <div class="flex-item">1</div>
  <div class="flex-item">2</div>
  <div class="flex-item">3</div>
</div>

Direction of Flex Items

Use the flex-direction property to set the direction of the flex items:

  • row (default): Items are placed in a row, from left to right.
  • row-reverse: Items are placed in a row, from right to left.
  • column: Items are placed in a column, from top to bottom.
  • column-reverse: Items are placed in a column, from bottom to top.

<style>
.flex-container {
  display: flex;
  flex-direction: row-reverse;
  background-color: lightcoral;
  padding: 10px;
}
.flex-item {
  background-color: lightgreen;
  padding: 20px;
  text-align: center;
  margin: 5px;
}
</style>
<div class="flex-container">
  <div class="flex-item">1</div>
  <div class="flex-item">2</div>
  <div class="flex-item">3</div>
</div>

Alignment Along the Main Axis

Use the justify-content property to control how flex items are aligned along the main axis:

  • flex-start (default): Items are aligned at the start of the main axis.
  • flex-end: Items are aligned at the end of the main axis.
  • center: Items are aligned in the center of the main axis.
  • space-between: Items are evenly distributed with space between them.
  • space-around: Items are evenly distributed with space around them.

<style>
.flex-container {
  display: flex;
  justify-content: space-around;
  background-color: lightyellow;
  padding: 10px;
}
.flex-item {
  background-color: lightpink;
  padding: 20px;
  text-align: center;
  margin: 5px;
}
</style>
<div class="flex-container">
  <div class="flex-item">A</div>
  <div class="flex-item">B</div>
  <div class="flex-item">C</div>
</div>

Alignment Along the Cross Axis

Use the align-items property to control how flex items are aligned along the cross axis:

  • stretch (default): Items stretch to fill the container.
  • flex-start: Items are aligned at the start of the cross axis.
  • flex-end: Items are aligned at the end of the cross axis.
  • center: Items are aligned at the center of the cross axis.
  • baseline: Items are aligned along their baseline.

<style>
.flex-container {
  display: flex;
  align-items: flex-end;
  background-color: lightblue;
  padding: 10px;
height: 150px;
}
.flex-item {
  background-color: lightgray;
  padding: 20px;
  text-align: center;
  margin: 5px;
}
</style>
<div class="flex-container">
  <div class="flex-item">X</div>
  <div class="flex-item">Y</div>
  <div class="flex-item">Z</div>
</div>

CSS Flex: Exercises

1. Basic Flex Layout

Create a flex container and three child elements. Apply the following styles to the container:

  • display: flex
  • Background color: lightgray
  • Padding: 20px
Style each child element with a lightblue background, 10px margin, and centered text. Observe how they align in a row.

<style>
.flex-container {
  display: flex;
  background-color: lightgray;
  padding: 20px;
}
.flex-item {
  background-color: lightblue;
  margin: 10px;
  text-align: center;
  padding: 10px;
}
</style>
<div class="flex-container">
  <div class="flex-item">1</div>
  <div class="flex-item">2</div>
  <div class="flex-item">3</div>
</div>

2. Changing Flex Direction

Create a flex container with five child elements. Style the container to use flex-direction: column and add a 15px gap between items. Observe how the items stack vertically.

<style>
.flex-container {
  display: flex;
  flex-direction: column;
  gap: 15px;
}
.flex-item {
  background-color: lightcoral;
  text-align: center;
  padding: 10px;
}
</style>
<div class="flex-container">
  <div class="flex-item">A</div>
  <div class="flex-item">B</div>
  <div class="flex-item">C</div>
  <div class="flex-item">D</div>
  <div class="flex-item">E</div>
</div>

3. Justifying Content

Create a flex container with four child elements. Style the container to:

  • Use justify-content: space-between
  • Set the background color to lightgreen
Observe how the items are evenly spaced along the main axis.

<style>
.flex-container {
  display: flex;
  justify-content: space-between;
  background-color: lightgreen;
  padding: 10px;
}
.flex-item {
  background-color: lightgray;
  text-align: center;
  padding: 10px;
}
</style>
<div class="flex-container">
  <div class="flex-item">1</div>
  <div class="flex-item">2</div>
  <div class="flex-item">3</div>
  <div class="flex-item">4</div>
</div>
