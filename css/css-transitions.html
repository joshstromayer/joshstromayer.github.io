CSS Transitions

CSS Transitions allow you to create smooth animations when an element changes from one state to another. You can transition properties like colors, sizes, and positions over a specified duration, making your designs more interactive and visually appealing.

Basic Syntax

A CSS Transition is defined using the transition property. The basic syntax includes:

  • Property: The CSS property you want to animate (e.g., background-color).
  • Duration: The time the transition takes (e.g., 0.5s).
  • Timing Function: The speed curve of the animation (e.g., ease, linear, ease-in, etc.).
  • Delay: The time to wait before starting the transition (optional).
<style>
div {
  width: 100px;
  height: 100px;
  background-color: lightblue;
  transition: background-color 0.5s ease;
}
div:hover {
  background-color: lightcoral;
}
</style>
<div></div>

In this example, the background color of the box smoothly changes when you hover over it.

Transitioning Multiple Properties

You can transition multiple properties by separating them with commas:

<style>
div {
  width: 100px;
  height: 100px;
  background-color: lightblue;
  border-radius: 0;
  transition: background-color 0.5s ease, border-radius 0.5s ease;
}
div:hover {
  background-color: lightgreen;
  border-radius: 50%;
}
</style>
<div></div>

Using Shorthand

The transition property is a shorthand for the individual transition properties:

  • transition-property: Specifies the property to animate.
  • transition-duration: Specifies the duration of the animation.
  • transition-timing-function: Specifies the speed curve.
  • transition-delay: Specifies the delay before the transition starts.
<style>
div {
  width: 100px;
  height: 100px;
  background-color: lightyellow;
  transition-property: background-color;
  transition-duration: 0.3s;
  transition-timing-function: ease-in;
  transition-delay: 0.2s;
}
div:hover {
  background-color: lightblue;
}
</style>
<div></div>

Timing Functions

The transition-timing-function determines how the speed of the animation changes over time. Common values include:

  • linear: The animation progresses at the same speed throughout.
  • ease: The animation starts slow, accelerates, and then slows down.
  • ease-in: The animation starts slow and speeds up.
  • ease-out: The animation starts fast and slows down.
  • ease-in-out: The animation starts slow, speeds up, and then slows down.
<style>
div {
  width: 100px;
  height: 100px;
  background-color: lightcoral;
  transition: background-color 1s ease-in-out;
}
div:hover {
  background-color: lightgreen;
}
</style>
<div></div>

Combining Transitions with Other Effects

CSS transitions can be combined with transformations like scaling or rotating to create more dynamic effects:

<style>
div {
  width: 100px;
  height: 100px;
  background-color: lightblue;
  transition: transform 0.5s ease, background-color 0.5s ease;
}
div:hover {
  background-color: lightpink;
  transform: scale(1.5);
}
</style>

CSS Transitions: Exercises

1. Basic Transition

Create a box with the following styles:

  • Width: 150px
  • Height: 150px
  • Background color: lightblue
Add a transition to change the background color to lightcoral when you hover over it. Make the transition last 0.5 seconds.

<style>
div {
  width: 150px;
  height: 150px;
  background-color: lightblue;
  transition: background-color 0.5s;
}
div:hover {
  background-color: lightcoral;
}
</style>
<div></div>

2. Transitioning Multiple Properties

Create a circular element with a 100px width and height. Add transitions to:

  • Change the background color to lightgreen when hovered.
  • Change the size to 150px by 150px when hovered.
Set the transition duration for both properties to 0.7 seconds.

<style>
div {
  width: 100px;
  height: 100px;
  background-color: lightblue;
  border-radius: 50%;
  transition: background-color 0.7s, width 0.7s, height 0.7s;
}
div:hover {
  background-color: lightgreen;
  width: 150px;
  height: 150px;
}
</style>
<div></div>

3. Timing Functions

Create a box that smoothly changes its background color from yellow to pink over 2 seconds when hovered. Use the ease-in-out timing function to make the animation more dynamic.

<style>
div {
  width: 200px;
  height: 100px;
  background-color: yellow;
  transition: background-color 2s ease-in-out;
}
div:hover {
  background-color: pink;
}
</style>
<div></div>

4. Combining Transformations

Create a square box that:

  • Scales to 1.5 times its size when hovered.
  • Rotates by 45 degrees during the same hover.
Use a transition duration of 1 second for both effects.

<style>
div {
  width: 100px;
  height: 100px;
  background-color: lightcoral;
  transition: transform 1s;
}
div:hover {
  transform: scale(1.5) rotate(45deg);
}
</style>
<div></div>

5. Transition Delay

Create a button that changes its background color to green when hovered. Add a 0.5-second delay before the transition begins, and make the transition last 1 second.

<style>
button {
  padding: 10px 20px;
  background-color: lightblue;
  border: none;
  transition: background-color 1s ease 0.5s;
}
button:hover {
  background-color: green;
}
</style>
