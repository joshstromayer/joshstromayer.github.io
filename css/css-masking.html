CSS Masking

CSS Masking allows you to hide or reveal parts of an element by applying a mask image or clip path. This is especially useful for creating complex visual effects and designs.

The mask Property

The mask property defines how an element is masked. Masks can use gradients, patterns, or images to define visible and hidden areas. The syntax is:

<style>
.masked {
  width: 300px;
  height: 200px;
  background-color: lightblue;
  mask-image: url('mask.png');
  -webkit-mask-image: url('mask.png');
}
</style>
<div class="masked"></div>

In this example, the element is masked using an image file, mask.png. The visible parts of the element correspond to the non-transparent parts of the mask image.

Using Gradients as Masks

Gradients can also be used as masks to create smooth transitions between visible and hidden areas:

<style>
.gradient-mask {
  width: 300px;
  height: 200px;
  background-color: lightcoral;
  mask-image: linear-gradient(to right, transparent, black);
  -webkit-mask-image: linear-gradient(to right, transparent, black);
}
</style>
<div class="gradient-mask"></div>

In this example, the mask gradually reveals the element from left to right.

Masking with Shapes

You can also use shapes, such as circles or polygons, as masks with the mask-image and clip-path properties:

<style>
.shape-mask {
  width: 300px;
  height: 300px;
  background-color: lightgreen;
  mask-image: radial-gradient(circle, black 50%, transparent 51%);
  -webkit-mask-image: radial-gradient(circle, black 50%, transparent 51%);
}
</style>
<div class="shape-mask"></div>

This creates a circular mask where only the center of the element is visible.

Mask Repeat, Position, and Size

Similar to background images, masks can have additional properties to control their appearance:

  • mask-repeat: Specifies whether the mask repeats (repeat, no-repeat, repeat-x, or repeat-y).
  • mask-position: Specifies the starting position of the mask.
  • mask-size: Specifies the size of the mask.
<style>
.repeated-mask {
  width: 300px;
  height: 200px;
  background-color: lightpink;
  mask-image: url('pattern.png');
  mask-repeat: repeat;
  -webkit-mask-image: url('pattern.png');
  -webkit-mask-repeat: repeat;
}
</style>
<div class="repeated-mask"></div>

In this example, the mask repeats across the entire element using the image pattern.png.

Masking Images

Apply a mask to an image for creative effects:

<style>
.masked-image {
  width: 300px;
  height: 300px;
  mask-image: radial-gradient(circle, black 50%, transparent 51%);
  -webkit-mask-image: radial-gradient(circle, black 50%, transparent 51%);
  overflow: hidden;
}
</style>
<img class="masked-image" src="image.jpg" alt="Masked Image">

This example uses a circular mask to hide the edges of the image.

Practical Applications

CSS masking is used in a variety of scenarios, including:

  • Creating visually appealing UI elements like buttons and cards.
  • Designing custom shapes and animations.
  • Hiding or highlighting specific areas of images or text.
