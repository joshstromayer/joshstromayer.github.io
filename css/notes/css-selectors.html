CSS Selectors

CSS selectors are patterns used to select and style specific elements on your web page. They are one of the most powerful tools in CSS, allowing you to target elements based on their type, class, ID, attributes, and more.

Types of Selectors

Below are some of the most commonly used CSS selectors:

  • Universal Selector (*): Selects all elements on the page.
  • Type Selector: Selects all elements of a specific type (e.g., h1, p).
  • Class Selector (.): Selects all elements with a specific class (e.g., .classname).
  • ID Selector (#): Selects the element with a specific ID (e.g., #idname).

Universal Selector

The universal selector applies styles to all elements on the page. For example:

* {
  margin: 0;
  padding: 0;
}

Type Selector

The type selector targets all elements of a specific type. For example:

p {
  color: blue;
}

Class Selector

The class selector targets elements with a specific class. Use a dot (.) followed by the class name. For example:

.highlight {
  background-color: yellow;
}

ID Selector

The ID selector targets an element with a specific ID. Use a hash symbol (#) followed by the ID name. For example:

#main-title {
  font-size: 24px;
  color: green;
}

Grouping Selectors

You can group selectors to apply the same styles to multiple elements. Use a comma to separate the selectors. For example:

h1, p, .highlight {
  text-align: center;
}

Combining Selectors

You can combine selectors to target elements more specifically. For example:

  • Descendant Selector: Targets elements nested inside others (e.g., div p selects all p elements inside a div).
  • Child Selector (>): Targets direct children of an element (e.g., div > p).
  • Adjacent Sibling Selector (+): Targets the next sibling element (e.g., h1 + p).
  • General Sibling Selector (~): Targets all siblings of an element (e.g., h1 ~ p).

We will cover these advanced selectors in a later topic.

Practical Example

Here's an example of using multiple selectors to style different parts of your web page:

body {
  font-family: Arial, sans-serif;
}

.header {
  background-color: lightblue;
}

#main-title {
  color: darkblue;
}

p {
  line-height: 1.6;
}

.container p {
  color: gray;
}

In this example:

  • body: Styles the entire page's font family.
  • .header: Targets an element with the class "header" and sets its background color.
  • #main-title: Styles an element with the ID "main-title" to change the text color.
  • p: Styles all <p> elements globally by setting the line height.
  • .container p: Demonstrates a descendant selector, targeting only <p> elements inside an element with the class "container" and setting their color to gray.

Descendant selectors are helpful when you want to style specific elements based on their location in the document hierarchy.

CSS selectors are incredibly powerful and allow you to style your web pages with great precision. Explore more advanced selectors in the future to unlock their full potential!

CSS Selectors: Exercises

1. Select All Paragraphs

Write CSS to change the font "color" of all paragraphs on the page to blue.

p {
  color: blue;
}

2. Target a Specific ID

Use CSS to set the "background-color" of the element with the ID #main-heading to yellow.

#main-heading {
  background-color: yellow;
}

3. Apply Styles to a Class

Create a CSS rule to make all elements with the class .highlight have italic text (use font-style) and a "font-size" of 18px.

.highlight {
  font-style: italic;
  font-size: 18px;
}

4. Combine Selectors

Write CSS to make all <h1> elements inside a <div> with the class .header have a text color of red.

.header h1 {
  color: red;
}

5. Group Selectors

Use CSS to apply the same background color of light gray to all <h2> and <p> elements on the page.

h2, p {
  background-color: lightgray;
}
