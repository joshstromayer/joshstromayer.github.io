CSS Display

The display property in CSS determines how an element is displayed on the webpage. It defines the layout behavior of the element and impacts its interaction with other elements. Common values for the display property include: block, inline, inline-block, none, flex, and grid.

Block Display

Elements with a display value of block take up the full width of their container and start on a new line. Examples of block elements include <div>, <p>, and <section>.

<style>
div {
  display: block;
  width: 100%;
  background-color: lightblue;
  padding: 10px;
}
</style>
<div>This is a block element.</div>
This is a block element.

Inline Display

Inline elements do not start on a new line and only take up as much width as necessary. Examples include <span>, <a>, and <strong>.

<style>
span {
  display: inline;
  background-color: lightcoral;
}
</style>
<p>This is an example of inline display.</p>

This is an example of inline display.

Inline-Block Display

Inline-block elements are similar to inline elements, but they allow setting width and height. They do not start on a new line.

<style>
div {
  display: inline-block;
  width: 100px;
  height: 50px;
  background-color: lightgreen;
  margin: 5px;
}
</style>

None Display

The none value hides an element from the webpage, removing it from the layout. The element does not occupy any space.

<style>
p.hidden {
  display: none;
}
</style>
<p>This paragraph is visible.</p>
<p class="hidden">This paragraph is hidden.</p>

This paragraph is visible.

This paragraph is hidden.

Flex Display

The flex value creates a flexible layout container. Its children are aligned according to the specified flexbox properties, such as justify-content and align-items.

<style>
.flex-container {
  display: flex;
}
.flex-item {
  width: 100px;
  height: 50px;
  background-color: lightyellow;
}
</style>
<div class="flex-container">
  <div class="flex-item">Item 1</div>
  <div class="flex-item">Item 2</div>
  <div class="flex-item">Item 3</div>
</div>

Grid Display

The grid value creates a grid-based layout. You can define rows, columns, and areas to organize elements.

<style>
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}
.grid-item {
  background-color: lightgray;
  text-align: center;
  padding: 10px;
}
</style>
<div class="grid-container">
  <div class="grid-item">1</div>
  <div class="grid-item">2</div>
  <div class="grid-item">3</div>
  <div class="grid-item">4</div>
  <div class="grid-item">5</div>
  <div class="grid-item">6</div>
</div>
