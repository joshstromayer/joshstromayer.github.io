CSS Units

In CSS, units are used to define the dimensions of elements, such as width, height, margin, padding, and font size. Understanding CSS units is crucial for creating responsive and flexible designs.

Types of CSS Units

CSS provides two main types of units:

  • Absolute Units: Fixed sizes that do not change regardless of the viewport or parent element.
  • Relative Units: Flexible sizes that depend on other elements, such as the viewport or parent element.

Absolute Units

Absolute units have fixed sizes and are not influenced by the viewport or parent element. They are best used when the exact size of an element needs to be defined.

  • px: Pixels. The most common unit for web design.
  • cm: Centimeters. Useful for print media.
  • mm: Millimeters. Useful for print media.
  • in: Inches. Primarily for print.
  • pt: Points. 1 point = 1/72 inch. Often used for fonts in print.
  • pc: Picas. 1 pica = 12 points.
<style>
div {
  width: 100px;
  height: 50px;
  background-color: lightblue;
}
</style>
<div>Fixed size with px</div>

Relative Units

Relative units adjust based on other elements, making them ideal for responsive designs.

  • em: Relative to the font size of the parent element.
  • rem: Relative to the root element's font size (usually the <html> element).
  • %: Percentage of the parent element's size.
  • vw: Percentage of the viewport width.
  • vh: Percentage of the viewport height.
  • vmin: The smaller value of vw and vh.
  • vmax: The larger value of vw and vh.
<style>
div {
  width: 50%;
  height: 20vh;
  background-color: lightgreen;
}
</style>
<div>Responsive size with % and vh</div>

Combining Units

You can combine different units to achieve the desired layout and responsiveness. For example:

<style>
div {
  width: calc(100% - 20px);
  height: 10rem;
  background-color: lightcoral;
}
</style>
<div>Combined units example</div>

When to Use Each Unit

  • Use px for precise control over dimensions.
  • Use em or rem for scalable typography.
  • Use % for relative sizes within a container.
  • Use vw and vh for full-page layouts.

1. Using Absolute Units

Create a <div> with the following properties:

  • Width: 5cm
  • Height: 2in
  • Background color: lightblue

  width: 5cm;
  height: 2in;
  background-color: lightblue;
}
</style>

2. Using Relative Units

Create a <div> with the following properties:

  • Width: 50% of the parent container
  • Height: 20vh
  • Background color: lightgreen
<style>
div {
  width: 50%;
  height: 20vh;
  background-color: lightgreen;
}
</style>
<div>Relative Units Example</div>

3. Using em and rem

Create a <p> element inside a <div> and apply the following styles:

  • The <div> should have a font size of 16px
  • The <p> should have a font size of 1.5em
  • The <p> text color should be red
<style>
div {
  font-size: 16px;
}
p {
  font-size: 1.5em;
  color: red;
}
</style>
<div>
  <p>Text with em units</p>
</div>

4. Combining Units

Create a <div> with the following properties:

  • A width of 50px less than 100% of the viewport
  • A height of 30rem
<style>
div {
  width: calc(100% - 50px);
  height: 30rem;
}
</style>
