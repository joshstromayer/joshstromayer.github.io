CSS Overflow

The overflow property in CSS controls what happens when the content of an element exceeds its container's size. It is commonly used to manage scrollbars or hide excess content.

Overflow Property Values

The overflow property can take the following values:

  • visible (default): The content overflows the container and is fully visible.
  • hidden: The content that overflows the container is clipped and not visible.
  • scroll: Adds scrollbars to the container to allow scrolling through the overflowing content.
  • auto: Adds scrollbars only if the content overflows the container.

Visible Overflow

By default, overflow is set to visible, meaning that the content will extend beyond the container without being clipped.

<style>
.visible-box {
  width: 150px;
  height: 100px;
  overflow: visible;
  background-color: lightblue;
  border: 1px solid black;
}
</style>
<div class="visible-box">
  This is some content that overflows the box.
  It will be visible outside the container.
</div>

Hidden Overflow

Setting overflow: hidden clips any content that extends beyond the container, preventing it from being displayed.

<style>
.hidden-box {
  width: 150px;
  height: 100px;
  overflow: hidden;
  background-color: lightcoral;
  border: 1px solid black;
}
</style>
<div class="hidden-box">
  This is some content that overflows the box.
  It will be clipped and not visible outside the container.
</div>

Scroll Overflow

Using overflow: scroll adds scrollbars to the container, allowing the user to scroll through the overflowing content.

<style>
.scroll-box {
  width: 150px;
  height: 100px;
  overflow: scroll;
  background-color: lightgreen;
  border: 1px solid black;
}
</style>
<div class="scroll-box">
  This is some content that overflows the box.
  Scrollbars will appear to let you view the hidden content.
</div>

Auto Overflow

The auto value adds scrollbars only if the content overflows the container. If the content fits within the container, no scrollbars will appear.

<style>
.auto-box {
  width: 150px;
  height: 100px;
  overflow: auto;
  background-color: lightyellow;
  border: 1px solid black;
}
</style>
<div class="auto-box">
  This is some content that overflows the box.
  Scrollbars will appear if needed.
</div>
