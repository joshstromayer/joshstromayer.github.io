CSS Fonts

Fonts in CSS allow you to control the appearance of text on your webpage. With CSS, you can define the font family, size, weight, style, and more to enhance the readability and design of your content.

Font Family

The font-family property specifies the typeface to be used for an element's text. You can list multiple fonts in order of preference as a fallback mechanism if the preferred font is unavailable.

<style>
p {
  font-family: Arial, Helvetica, sans-serif;
}
</style>
<p>This text uses Arial font or a fallback sans-serif font.</p>

Font Size

The font-size property controls the size of the text. You can define font sizes using absolute units (e.g., px, pt) or relative units (e.g., em, rem, %).

<style>
h1 {
  font-size: 32px;
}
p {
  font-size: 1.2em;
}
</style>
<h1>Large Heading</h1>
<p>This is slightly larger text using relative units.</p>

Font Weight

The font-weight property adjusts the thickness of the text. Common values include normal, bold, and numeric weights like 100 (thin) to 900 (extra-bold).

<style>
h1 {
  font-weight: bold;
}
p {
  font-weight: 300;
}
</style>
<h1>Bold Heading</h1>
<p>This is light-weight text.</p>

Font Style

The font-style property is used to make text italicized or oblique. Common values are normal, italic, and oblique.

<style>
em {
  font-style: italic;
}
</style>
<p>This text is <em>italicized</em>.</p>

Line Height

The line-height property controls the space between lines of text, improving readability. It is commonly defined as a unitless number or a percentage.

<style>
p {
  line-height: 1.5;
}
</style>
<p>This paragraph has increased line spacing for better readability.</p>

Combining Font Properties

The font shorthand property allows you to define multiple font-related properties in a single line.

<style>
p {
  font: italic bold 16px/1.5 Georgia, serif;
}
</style>
<p>This text combines multiple font properties.</p>

CSS Fonts: Exercises

1. Changing Font Family

Style a <p> element to use the font family "Arial, sans-serif".

<style>
p {
  font-family: Arial, sans-serif;
}
</style>
<p>This text is in Arial.</p>

2. Setting Font Size

Create a <h1> element with a font size of 32px.

<style>
h1 {
  font-size: 32px;
}
</style>
<h1>This is a heading.</h1>

3. Font Weight

Create a <p> element with bold text using the font-weight property.

<style>
p {
  font-weight: bold;
}
</style>
<p>This text is bold.</p>

4. Font Style

Create a <p> element with italicized text.

<style>
p {
  font-style: italic;
}
</style>
<p>This text is italicized.</p>

5. Combining Font Properties

Style a <div> to use the following font properties:

  • Font family: "Georgia, serif"
  • Font size: 18px
  • Font weight: 700
  • Font style: italic

<style>
div {
  font-family: Georgia, serif;
  font-size: 18px;
  font-weight: 700;
  font-style: italic;
}
</style>
<div>This text combines font properties.</div>
