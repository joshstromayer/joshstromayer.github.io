CSS Text

CSS provides a variety of properties to style and manipulate text. From changing the color of the text to adjusting spacing, alignment, and decoration, CSS makes text styling easy and powerful.

Text Color

You can use the color property to change the color of text. The value can be a predefined color name, a hexadecimal value, an RGB value, or an HSL value.

<style>
p {
  color: blue;
}
</style>
<p>This text is blue.</p>

Text Alignment

Use the text-align property to align text within an element. Common values include left, right, center, and justify.

<style>
h1 {
  text-align: center;
}
</style>
<h1>This text is centered.</h1>

Text Decoration

The text-decoration property is used to add or remove decorations like underlines, overlines, or line-throughs (strikethroughs) from text.

<style>
a {
  text-decoration: none;
}
</style>
<a href="#">This is a link without an underline.</a>

Text Transformation

The text-transform property controls the capitalization of text. You can use values like uppercase, lowercase, or capitalize.

<style>
p {
  text-transform: uppercase;
}
</style>
<p>This text is uppercase.</p>

Line Height

The line-height property adjusts the spacing between lines of text. It’s commonly used to improve readability.

<style>
p {
  line-height: 1.8;
}
</style>
<p>This text has increased line spacing.</p>

Letter Spacing

Use the letter-spacing property to control the space between characters in a text. Positive values increase the spacing, while negative values decrease it.

<style>
h2 {
  letter-spacing: 2px;
}
</style>
<h2>This text has spaced letters.</h2>

Word Spacing

The word-spacing property adjusts the spacing between words in a text. Like letter-spacing, positive values increase the space, and negative values decrease it.

<style>
p {
  word-spacing: 5px;
}
</style>
<p>This text has extra spacing between words.</p>

CSS Text: Exercises

1. Change Text Color

Set the text color of a <p> element to green.

<style>
p {
  color: green;
}
</style>
<p>This text is green.</p>

2. Center Align Text

Create an <h1> element and center align its text.

<style>
h1 {
  text-align: center;
}
</style>
<h1>This text is centered.</h1>

3. Remove Underline from Links

Create a link (<a> element) and remove its underline.

<style>
a {
  text-decoration: none;
}
</style>
<a href="#">This is a link without an underline.</a>

4. Capitalize Text

Style a <p> element so its text is displayed in all uppercase letters.

<style>
p {
  text-transform: uppercase;
}
</style>
<p>THIS TEXT IS UPPERCASE.</p>

5. Adjust Line Height

Create a <p> element with text and set its line height to 2.

<style>
p {
  line-height: 2;
}
</style>
<p>This text has a line height of 2.</p>

6. Add Letter Spacing

Create an <h2> element and add 4px of letter spacing.

<style>
h2 {
  letter-spacing: 4px;
}
</style>
<h2>Spaced Letters</h2>

7. Combine Multiple Properties

Style a <p> element with the following:

  • Text color: blue
  • Text alignment: right
  • Text decoration: underline
<style>
p {
  color: blue;
  text-align: right;
  text-decoration: underline;
}
</style>
<p>Styled Text</p>
