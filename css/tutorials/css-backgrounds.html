CSS Backgrounds

CSS backgrounds are a powerful way to enhance the look and feel of your webpage. You can style backgrounds with colors, images, gradients, and even patterns, allowing for greater design flexibility and creativity.

Background Color

The background-color property sets the background color of an element. You can use color names, HEX values, RGB, RGBA, or HSL values.

<style>
div {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}
</style>
<div>This div has a light blue background.</div>

Background Image

Use the background-image property to set an image as the background of an element. You can specify the image using the url() function.

<style>
div {
  background-image: url('background.jpg');
  background-size: cover;
  height: 200px;
}
</style>
<div></div>

Background Size

The background-size property specifies the size of the background image. Common values include:

  • cover: Ensures the background image covers the entire element.
  • contain: Ensures the background image fits within the element.
  • Custom sizes like 50% or 100px.
<style>
div {
  background-image: url('background.jpg');
  background-size: cover;
  height: 300px;
}
</style>
<div></div>

Background Repeat

The background-repeat property controls whether the background image is repeated. Values include:

  • repeat: Default value; repeats the image.
  • no-repeat: Prevents the image from repeating.
  • repeat-x: Repeats the image horizontally.
  • repeat-y: Repeats the image vertically.
<style>
div {
  background-image: url('pattern.png');
  background-repeat: repeat-x;
  height: 200px;
}
</style>
<div></div>

Background Position

The background-position property specifies the starting position of a background image. Values can include keywords (e.g., center, top left) or coordinates (e.g., 50% 50% or 10px 20px).

<style>
div {
  background-image: url('image.jpg');
  background-position: center;
  height: 200px;
}
</style>
<div></div>

Background Attachment

The background-attachment property determines whether the background image scrolls with the page or remains fixed:

  • scroll: The background image scrolls with the content (default).
  • fixed: The background image remains fixed when scrolling.
<style>
div {
  background-image: url('background.jpg');
  background-attachment: fixed;
  height: 300px;
}
</style>
<div></div>

CSS Backgrounds: Exercises

1. Add a Background Color

Set the background color of a <div> to lightgreen and add some text inside it. The text should be clear and visible.

<style>
div {
  background-color: lightgreen;
  height: 100px;
  color: red; /* color must not be similar to lightgreen */
}
</style>
<div>This is a lightgreen background</div>

2. Add a Background Image

Set an image as the background of a <div>. Make sure the image covers the entire <div> and does not repeat.

<style>
div {
  background-image: url('background.jpg');
  background-size: cover;
  background-repeat: no-repeat;
  height: 200px;
}
</style>
<div></div>

3. Use Background Position

Create a <div> with a background image and position it at the bottom-right corner of the <div>.

<style>
div {
  background-image: url('image.jpg');
  background-position: bottom right;
  height: 150px;
  width: 300px;
}
</style>
<div></div>

4. Create a Fixed Background

Create a <div> with a height of 500px and set a fixed background image that does not scroll with the content.

<style>
div {
  background-image: url('fixed-background.jpg');
  background-attachment: fixed;
  background-size: cover;
  height: 500px;
}
</style>
<div></div>

5. Combine Background Properties

Create a <div> with the following background properties:

  • A background image of your choice
  • Centered horizontally and vertically
  • No repetition
  • Covers the entire <div>

<style>
div {
  background-image: url('example.jpg');
  background-size: cover;
  background-position: center;
  background-repeat: no-repeat;
  height: 250px;
}
</style>
<div></div>
