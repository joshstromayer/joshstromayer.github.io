CSS Grid

The grid property in CSS is used to create two-dimensional layouts on the web. It allows you to design layouts with rows and columns, providing greater control over the placement of elements.

Setting Up a Grid

To create a grid layout, you use the display: grid property on a container element. Child elements within the grid container automatically become grid items.

<style>
.grid-container {
  display: grid;
  grid-template-columns: 100px 100px 100px;
  gap: 10px;
}
.grid-item {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}
</style>
<div class="grid-container">
  <div class="grid-item">1</div>

Grid Template Columns and Rows

The grid-template-columns and grid-template-rows properties define the number and size of columns and rows in the grid. You can specify sizes in pixels, percentages, or the fr unit, which stands for "fraction of available space."

<style>
.grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  grid-template-rows: 50px 100px;
  gap: 10px;
}
.grid-item {
  background-color: lightcoral;
  padding: 10px;
  text-align: center;
}
</style>
<div class="grid-container">
  <div class="grid-item">A</div>
  <div class="grid-item">B</div>
  <div class="grid-item">C</div>
  <div class="grid-item">D</div>
</div>

Placing Items in the Grid

You can control the placement of grid items using the grid-column and grid-row properties. These properties define where an item starts and how many columns or rows it spans.

<style>
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: repeat(2, 100px);
  gap: 10px;
}
.grid-item:nth-child(1) {
  grid-column: 1 / 3;
  grid-row: 1;
}
.grid-item:nth-child(2) {
  grid-column: 3;
  grid-row: 1 / 3;
}
.grid-item {
  background-color: lightgreen;
  text-align: center;
}
</style>
<div class="grid-container">
  <div class="grid-item">1</div>
  <div class="grid-item">2</div>
  <div class="grid-item">3</div>
  <div class="grid-item">4</div>
</div>

Gap Property

The gap property defines the space between rows and columns in a grid. You can set a uniform gap or specify different values for rows and columns using row-gap and column-gap.

<style>
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px 10px;
}
.grid-item {
  background-color: lightpink;
  text-align: center;
}
</style>
<div class="grid-container">
  <div class="grid-item">1</div>
  <div class="grid-item">2</div>
  <div class="grid-item">3</div>
  <div class="grid-item">4</div>
  <div class="grid-item">5</div>
  <div class="grid-item">6</div>
