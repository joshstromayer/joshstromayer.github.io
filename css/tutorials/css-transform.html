CSS Transform

The transform property in CSS allows you to apply 2D and 3D transformations to elements. You can move, rotate, scale, and skew elements to create dynamic and visually engaging designs.

Basic Syntax

The transform property accepts transformation functions as its value. For example:

<style>
div {
  width: 100px;
  height: 100px;
  background-color: lightblue;
  transform: rotate(45deg);
}
</style>
<div></div>

In this example, the element is rotated 45 degrees clockwise.

Transform Functions

Common 2D transform functions include:

  • translate(x, y): Moves the element along the X and Y axes.
  • rotate(angle): Rotates the element by the specified angle (e.g., 45deg).
  • scale(x, y): Resizes the element horizontally and vertically.
  • skew(x-angle, y-angle): Skews the element along the X and Y axes.

Translate

Use translate to move an element:

<style>
div {
  width: 100px;
  height: 100px;
  background-color: lightcoral;
  transform: translate(50px, 20px);
}
</style>
<div></div>

This moves the element 50px to the right and 20px down.

Rotate

Use rotate to rotate an element:

<style>
div {
  width: 100px;
  height: 100px;
  background-color: lightgreen;
  transform: rotate(30deg);
}
</style>
<div></div>

This rotates the element 30 degrees clockwise.

Scale

Use scale to resize an element:

<style>
div {
  width: 100px;
  height: 100px;
  background-color: lightblue;
  transform: scale(1.5, 2);
}
</style>
<div></div>

This scales the element to 1.5 times its original width and 2 times its original height.

Skew

Use skew to distort an element:

<style>
div {
  width: 100px;
  height: 100px;
  background-color: lightyellow;
  transform: skew(20deg, 10deg);
}
</style>
<div></div>

This skews the element 20 degrees along the X-axis and 10 degrees along the Y-axis.

Combining Transforms

You can combine multiple transformations in a single transform property:

<style>
div {
  width: 100px;
  height: 100px;
  background-color: lightpink;
  transform: translate(50px, 20px) rotate(45deg) scale(1.2);
}
</style>
<div></div>

This moves the element, rotates it, and scales it simultaneously.
