CSS Width and Height

The width and height properties in CSS are used to define the dimensions of an element. These properties are essential for controlling the size of block-level and inline elements.

Setting Width and Height

You can set the width and height of an element using various units such as pixels (px), percentages (%), or other unit types like em and rem.

<style>
div {
  width: 300px;
  height: 150px;
  background-color: lightblue;
}
</style>
<div>This is a box</div>

The above code creates a box with a width of 300px and a height of 150px.

Using Percentages

When using percentages, the width and height are calculated relative to the parent element's dimensions.

<style>
div {
  width: 50%;
  height: 100px;
  background-color: lightgreen;
}
</style>
<div>This box is 50% of the parent width</div>

Min and Max Width/Height

CSS provides the min-width, max-width, min-height, and max-height properties to set constraints on an element's size. These properties prevent the element from becoming too small or too large.

<style>
div {
  width: 100%;
  max-width: 600px;
  height: 200px;
  background-color: lightcoral;
}
</style>
<div>This box will not exceed 600px</div>

Auto Value

The auto value allows the browser to calculate the width or height based on the element's content or other factors, such as margins.

<style>
div {
  width: auto;
  height: auto;
  background-color: lightyellow;
}
</style>
<div>This box adjusts to its content</div>

Box Sizing

The box-sizing property affects how the width and height of an element are calculated. The two most common values are:

  • content-box (default): Width and height apply only to the content, excluding padding and borders.
  • border-box: Width and height include content, padding, and borders.
<style>
div {
  width: 300px;
  height: 150px;
  box-sizing: border-box;
  padding: 20px;
  border: 5px solid black;
  background-color: lightgray;
}
</style>
<div>Border-Box Example</div>

CSS Width and Height: Exercises

1. Fixed Width and Height

Create a <div> with the following properties:

  • Width: 300px
  • Height: 150px
  • Background color: lightblue
<style>
div {
  width: 300px;
  height: 150px;
  background-color: lightblue;
}
</style>
<div>This is a fixed size box</div>

2. Percentage-Based Dimensions

Create a <div> with the following properties:

  • Width: 50% of the viewport
  • Height: 25% of the viewport
  • Background color: lightgreen
<style>
div {
  width: 50%;
  height: 25vh;
  background-color: lightgreen;
}
</style>
<div>Percentage dimensions</div>

3. Minimum and Maximum Dimensions

Create a <div> with the following properties:

  • Minimum Width: 200px
  • Minimum Height: 100px
  • Maximum Width: 400px
  • Maximum Height: 300px
  • Background color: lightcoral
Resize the browser to observe how the box behaves.

<style>
div {
  min-width: 200px;
  min-height: 100px;
  max-width: 400px;
  max-height: 300px;
  background-color: lightcoral;
}
</style>
<div>This box has min and max dimensions</div>

4. Responsive Square

Create a <div> that is always a square and scales with the viewport width.

  • Width: 20% of the viewport
  • Height: Same as the width
  • Background color: lightyellow

<style>
div {
  width: 20vw;
  height: 20vw;
  background-color: lightyellow;
}
</style>
