CSS Margins

Margins are used to create space around elements, outside of their borders. With margins, you can control the spacing between elements and create visually appealing layouts.

What Are Margins?

A margin is the space between an element's border and its surrounding elements. Margins are completely transparent and do not have any background or visible structure.

<style>
div {
  border: 1px solid black;
  margin: 20px;
}
</style>
<div>This is a div with a 20px margin.</div>

Setting Margins

Margins can be set using the margin property. You can specify margins in multiple ways:

  • Single Value: Applies the same margin to all four sides.
  • Two Values: The first value applies to the top and bottom margins, and the second value applies to the left and right margins.
  • Four Values: Apply margins in this order: top, right, bottom, and left.
<style>
div {
  margin: 10px; /* Single value */
  margin: 10px 20px; /* Two values */
  margin: 10px 15px 20px 25px; /* Four values */
}
</style>

Margin Shorthand

You can use shorthand to set all four margins in one declaration. The order for the values is:

  1. Top
  2. Right
  3. Bottom
  4. Left

For example:

<style>
div {
  margin: 10px 15px 20px 25px;
}
</style>

Auto Margins

Margins can also be set to auto. This is particularly useful for centering elements horizontally within their container.

<style>
div {
  width: 50%;
  margin: 0 auto;
  border: 1px solid black;
}
</style>
<div>This div is horizontally centered.</div>

Negative Margins

Margins can take negative values to pull elements closer together or even overlap them.

<style>
div {
  margin: -10px;
  border: 1px solid black;
}
</style>
<div>This div has a negative margin.</div>

Practical Example

Below is an example of how margins can be used to create spacing between elements:

<style>
.box {
  border: 1px solid black;
  margin: 20px;
  padding: 10px;
}
</style>
<div class="box">Box 1</div>
<div class="box">Box 2</div>

CSS Margins: Exercises

1. Add Margins to a Paragraph

Create a paragraph with 20px margin on all sides. Add some text inside the paragraph.

<style>
p {
  margin: 20px;
}
</style>
<p>This paragraph has a margin of 20px on all sides.</p>

2. Add Different Margins

Create a div with different margins: 10px on the top, 15px on the right, 20px on the bottom, and 25px on the left.

<style>
div {
  margin: 10px 15px 20px 25px;
  border: 1px solid black;
}
</style>
<div>This div has different margins.</div>

3. Center a Div with Margins

Use margins to horizontally center a div with a width of 200px.

<style>
div {
  width: 200px;
  margin: 0 auto;
  border: 1px solid black;
  text-align: center;
}
</style>
<div>This div is horizontally centered.</div>

4. Add Vertical Margins

Create two div elements stacked vertically with 30px margin between them.

<style>
.box {
  margin-bottom: 30px;
  border: 1px solid black;
  height: 50px;
}
</style>
<div class="box">First div</div>
<div class="box">Second div</div>
