CSS Shapes

CSS Shapes allow you to define geometric shapes for content wrapping around elements. This feature enables text and other inline content to flow naturally around custom shapes, creating visually engaging layouts.

The shape-outside Property

The shape-outside property defines the shape around which text content wraps. It works with floated elements and accepts various values, including:

  • circle(): Defines a circular shape.
  • ellipse(): Defines an elliptical shape.
  • polygon(): Defines a custom polygonal shape.
  • inset(): Defines a rectangular shape with optional offsets and rounded corners.

Using shape-outside with Shapes

1. Circle

Wrap text around a circular shape:

<style>
.circle {
  width: 150px;
  height: 150px;
  background-color: lightblue;
  float: left;
  shape-outside: circle(50%);
  clip-path: circle(50%);
  margin: 20px;
}
</style>

Practical Applications

CSS Shapes can be used for:

  • Creating visually engaging layouts where text flows naturally around shapes.
  • Highlighting content by wrapping it around design elements.
  • Creating magazine-like layouts or unique web designs.
