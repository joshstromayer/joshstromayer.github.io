HTML Paragraphs

Paragraphs in HTML are defined using the <p> tag. Paragraphs help structure text content, making it easier to read and organize. Each paragraph you write will automatically be separated by some space in most browsers, ensuring clear readability.

Creating a Paragraph

To create a paragraph in HTML, you simply wrap your text with <p> and </p> tags. Here is an example:

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

The above code will result in two distinct paragraphs on the web page, each with a space between them, as shown below:


This is a paragraph.

This is another paragraph.


Spaces and Line Breaks

In HTML, extra spaces or line breaks in your code do not affect the display of your text. For example, even if you add multiple spaces or write text on separate lines, the browser will treat it as a single space. Consider this code:

<p>This   is a     paragraph.</p>
<p>This
is written
on multiple
lines.</p>

The browser will render the above code as:

  • Single spaces between words, even if you include extra spaces in your code.
  • Line breaks will only appear if you explicitly use the <br> tag.

Forcing Line Breaks

To force a line break within a paragraph, use the <br> tag. Unlike <p>, which creates a new paragraph with extra space, <br> only breaks the line:

<p>This is the first line.<br>This is the second line.<br>And this is the third.</p>

The browser will display:

This is the first line.

This is the second line.

And this is the third.

Tips for Using Paragraphs

  • Keep it concise: Break your text into multiple paragraphs for better readability.
  • Use semantic structure: Use <p> for paragraphs, not other tags like <div>.
  • Combine with CSS: Style your paragraphs using CSS for font, color, and spacing adjustments.
