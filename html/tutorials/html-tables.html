HTML Tables

Tables in HTML are used to organize and display data in rows and columns. They are helpful for presenting structured information like schedules, pricing details, or comparisons.

Basic Table Structure

To create a table in HTML, use the <table> element. A table is made up of rows (<tr>) and columns (<td> for data cells and <th> for header cells).

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Stevie</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td>30</td>
  </tr>
</table>

Output:

Name Age
Stevie 25
Jane 30

Table Headers

Header cells are created using the <th> tag. They make the table more meaningful and improve accessibility.

Adding Borders

To add borders to your table, you can use the border attribute (not recommended in modern HTML) or CSS for styling. Here's an example:

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Stevie</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td>30</td>
  </tr>
</table>

Output:

Name Age
Stevie 25
Jane 30

Spanning Rows and Columns

You can use the rowspan and colspan attributes to make a cell span across multiple rows or columns.

<table border="1">
  <tr>
    <th>Name</th>
    <th colspan="2">Details</th>
  </tr>
  <tr>
    <td>Stevie</td>
    <td>Age: 25</td>
    <td>City: New York</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td colspan="2">Age: 30, City: LA</td>
  </tr>
</table>

Output:

Name Details
Stevie Age: 25 City: New York
Jane Age: 30, City: LA

Accessibility in Tables

Adding captions (<caption>) and summaries improves accessibility. Screen readers can use these elements to describe the content of the table.

<table border="1">
  <caption>Student Information</caption>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Stevie</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td>30</td>
  </tr>
</table>

Output:

Student Information
Name Age
Stevie 25
Jane 30

HTML Tables: Exercises

1. Create a Simple Table

Create a table with the following details:

  • A header row with columns: Product and Price.
  • Two rows: one for a "Laptop - $1200" and another for a "Phone - $800".

<table border="1">
  <tr>
    <th>Product</th>
    <th>Price</th>
  </tr>
  <tr>
    <td>Laptop</td>
    <td>$1200</td>
  </tr>
  <tr>
    <td>Phone</td>
    <td>$800</td>
  </tr>
</table>
Product Price
Laptop $1200
Phone $800

2. Add a Caption

Modify the previous table to include a caption that says: "Product Price List".

<table border="1">
  <caption>Product Price List</caption>
  <tr>
    <th>Product</th>
    <th>Price</th>
  </tr>
  <tr>
    <td>Laptop</td>
    <td>$1200</td>
  </tr>
  <tr>
    <td>Phone</td>
    <td>$800</td>
  </tr>

Product Price

3. Use Rowspan

Create a table for a weekly schedule with two columns: Day and Activity. Use rowspan to merge cells for the same activity on Monday and Tuesday.

<table border="1">
  <tr>
    <th>Day</th>
    <th>Activity</th>
  </tr>
  <tr>
    <td rowspan="2">Monday, Tuesday</td>
    <td>Study</td>
  </tr>
  <tr>
    <td>Exercise</td>
  </tr>
</table>
Day Activity
Monday, Tuesday Study
Exercise

4. Use Colspan

Create a table for a monthly expense report with three columns: Category, Amount, and Total. Use colspan to merge the header cells of the last column.

<table border="1">
  <tr>
    <th>Category</th>
    <th>Amount</th>
    <th colspan="2">Total</th>
  </tr>
  <tr>
    <td>Food</td>
    <td>$300</td>
    <td>$500</td>
  </tr>
</table>
Category Amount Total
Food $300 $500
Groceries $200
