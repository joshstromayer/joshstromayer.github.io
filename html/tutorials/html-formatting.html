HTML Formatting

Formatting in HTML allows you to style and emphasize text to make it more visually appealing and semantically meaningful. In this lesson, weâ€™ll cover several key formatting tags and their usage.

1. Bold and Strong Text

Use <b> to make text bold and <strong> to indicate important text. While both visually bolden the text, <strong> also has semantic importance, which helps screen readers emphasize the content.

This text is bold using <b>.
This text is bold and important using <strong>.

2. Italicized and Emphasized Text

Use <i> to italicize text and <em> to add emphasis. Similar to <b> and <strong>, <em> carries semantic meaning.

This text is italicized using <i>.
This text is emphasized using <em>.

3. Highlighted and Underlined Text

Use <mark> to highlight text and <u> to underline it. Highlighting is often used to grab attention, while underlining is more traditional.

This text is highlighted using <mark>.
This text is underlined using <u>.

4. Monospaced (Teletype) Text

The <tt> tag is used to display text in a monospaced font, similar to how code snippets are styled.

This text is displayed in a monospaced font using <tt>.

5. Strikethrough and Deleted Text

Use <strike> or <del> to indicate that text is no longer valid. <strike> is deprecated, so use <del> for modern HTML.

This text is struck through using <strike> (deprecated).
This text is struck through using <del>.

6. Inserted Text

Use <ins> to indicate new or added text.

This text is newly added using <ins>. Your browser would normally underline ins text.

7. Superscript and Subscript

Use <sup> for superscript (e.g., exponents) and <sub> for subscript (e.g., chemical formulas).

E = mc2 (Superscript)
H2O (Subscript)

8. Big and Small Text

Use <big> to enlarge text and <small> to shrink it slightly.

This text is larger using <big>.
This text is smaller using <small>.

Summary

Formatting tags help style and emphasize text for better readability and meaning. While some tags like <strike> are deprecated, others like <strong> and <em> offer both visual and semantic benefits. Remember to use formatting tags appropriately based on the content and context.
