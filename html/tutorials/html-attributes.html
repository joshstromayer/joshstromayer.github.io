HTML Attributes

HTML attributes provide additional information about HTML elements. They modify the default behavior of elements and allow you to customize them by adding extra functionality or data.

What Are Attributes?

Attributes are always written in the opening tag of an HTML element. They consist of a name and a value, separated by an equals sign (=). The value is enclosed in quotes. For example:

<a href="https://example.com">Visit Example</a>

In this example, the href attribute specifies the URL the link should navigate to.

Commonly Used Attributes

  • href: Used in <a> tags to specify the URL of a link.
  • src: Used in <img> and <script> tags to specify the source of an image or script.
  • alt: Provides alternative text for images when the image cannot be displayed.
  • id: Assigns a unique identifier to an element for styling or scripting.
  • class: Assigns a class name to an element for styling or grouping.
  • style: Adds inline CSS to an element.
  • title: Adds a tooltip that appears when the user hovers over the element.

Examples of Attributes in Action

1. The href Attribute

<a href="https://www.stromayer.com">Learn HTML</a>

This creates a hyperlink that navigates to CodeItClear when clicked.

2. The src and alt Attributes

<img src="image.jpg" alt="A description of the image">

The src attribute specifies the image file location, and alt provides alternative text if the image cannot load.

3. The style Attribute

<p style="color: blue;">This text is blue.</p>

The style attribute applies inline CSS, such as changing the text color to blue.

Important Notes About Attributes

  • Case-insensitivity: Attribute names are not case-sensitive, but lowercase is preferred for consistency.
  • Quotation marks: Attribute values should always be enclosed in double or single quotes.
  • Placement: Attributes must be included in the opening tag of an element.

HTML Attributes: Exercises

1. Create a Link

Write an HTML snippet to create a link that navigates to "https://example.com" and displays the text "Visit Example".

<a href="https://example.com">Visit Example</a>

2. Add an Image

Write an HTML snippet to display an image with the source "image.jpg" and the alternative text "A beautiful scenery".

<img src="image.jpg" alt="A beautiful scenery">

3. Style a Paragraph

Write an HTML snippet to create a paragraph with the text "Hello, World!" that is styled with a red color using the style attribute.

<p style="color: red;">Hello, World!</p>

4. Use Multiple Attributes

Write an HTML snippet to create an image with the source "logo.png", alternative text "Company Logo", and assign it the ID "logo-image".

<img src="logo.png" alt="Company Logo" id="logo-image">

5. Add a Tooltip

Write an HTML snippet to create a paragraph with the text "Hover over me!" and add a tooltip that says "This is a tooltip" when the user hovers over it.

<p title="This is a tooltip">Hover over me!</p>
Output:

Hover over me!
