HTML Classes

Classes in HTML are used to group elements that share the same styling or behavior. By assigning a class to one or more elements, you can target them collectively using CSS or JavaScript. Classes make it easy to create reusable styles and behaviors across your webpage.

What is a Class?

A class is an attribute that you can add to an HTML element. It allows you to group multiple elements under the same name for styling or scripting purposes. Classes are defined using the class attribute and can be applied to any HTML element.

<p class="highlight">This paragraph has a class.</p>
<div class="highlight">This div has the same class.</div>

Adding Classes to HTML Elements

To assign a class to an element, use the class attribute followed by the name of the class in quotation marks. You can apply the same class to multiple elements to group them together.

<h1 class="title">Welcome to My Website</h1>
<p class="description">This is the first paragraph.</p>
<p class="description">This is another paragraph with the same class.</p>

Using Multiple Classes

You can assign multiple classes to a single element by separating the class names with a space. This allows you to combine different styles or behaviors for the same element.

<div class="box rounded shadow">
  This div has multiple classes.
</div>

Why Use Classes?

Classes are incredibly useful for:

  • Applying consistent styling to multiple elements.
  • Grouping elements for JavaScript interactions.
  • Creating reusable styles for easier maintenance.

Difference Between Classes and IDs

While both classes and IDs can be used to identify elements, they serve different purposes:

  • Classes: Can be applied to multiple elements and are used for grouping and styling.
  • IDs: Should be unique within a page and are used to target a single, specific element.

<div id="unique-container">This div has an ID.</div>
<p class="common-style">This paragraph has a class.</p>
<p class="common-style">This paragraph shares the same class.</p>

Best Practices for Classes

When using classes, keep these best practices in mind:

  • Use meaningful class names that describe the purpose of the element (e.g., header, footer).
  • Avoid overly generic names like box or container unless the class is truly generic.
  • Use kebab-case (e.g., main-header) for class names to maintain consistency and readability.

HTML Classes: Exercises

1. Create and Apply a Class

Create a paragraph with the class "highlight". Try use CSS to change the text color of the class to blue.

<style>
  .highlight { color: blue; }
</style>

<p class="highlight">This text is blue.</p>

2. Apply Multiple Classes

Create a div with the classes "box" and "shadow". Try use CSS to give "box" a border and "shadow" a shadow effect.

<style>
  .box { border: 2px solid black; padding: 10px; }
  .shadow { box-shadow: 2px 2px 5px gray; }
</style>

<div class="box shadow">This div has a border and shadow.</div>

3. Reuse a Class

Create three paragraphs and apply the same class, "description", to all of them. Try use CSS to change the font size of the class to 18px.

<style>
  .description { font-size: 18px; }
</style>

<p class="description">Paragraph 1.</p>
<p class="description">Paragraph 2.</p>
<p class="description">Paragraph 3.</p>

4. Combine Classes

Create a div with the classes "highlight" and "rounded". Try use CSS to set the text color for "highlight" to red and add rounded corners for "rounded".

<style>
  .highlight { color: red; }
  .rounded { border-radius: 10px; padding: 10px; border: 1px solid black; }
</style>

<div class="highlight rounded">This div has red text and rounded corners.</div>

5. Differentiate Classes and IDs

Create two divs: one with the class "container" and another with the ID "main-container". Try use CSS to give the class a blue border and the ID a green background.

<style>
  .container { border: 2px solid blue; padding: 10px; }
  #main-container { background-color: lightgreen; padding: 10px; }
</style>

<div class="container">This div has a blue border.</div>
<div id="main-container">This div has a green background.</div>
