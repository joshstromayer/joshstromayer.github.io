HTML Links

Links are one of the most important elements in HTML, allowing users to navigate between pages, documents, or external websites. They are created using the <a> (anchor) tag.

Basic Syntax of Links

A basic link requires an href attribute that specifies the URL to navigate to. Here’s the general structure:

<a href="https://example.com">Visit Example</a>

When this code is rendered in a browser, it creates a clickable link with the text "Visit Example." Clicking on the link will take you to https://example.com.

Opening Links in a New Tab

To open a link in a new tab, add the target="_blank" attribute to the anchor tag:

<a href="https://example.com" target="_blank">Visit Example</a>

The target="_blank" attribute instructs the browser to open the link in a new tab or window, depending on the browser settings.

Linking to Other Pages in Your Website

When linking to other pages within your website, you can use relative file paths. For example:

<a href="about.html">About Us</a>

This link assumes that the file about.html is in the same directory as the current file.

You can also link to files located in different folders or directories, as long as they are accessible from the current file's location:

<a href="./images/logo.png">View Logo</a>
<a href="../styles/styles.css">View Stylesheet</a>

Explanation:

  • ./images/logo.png: Refers to the images folder in the current directory.
  • ../styles/styles.css: Refers to the css folder one level above the current directory.

While this approach works, it is generally good practice to organize your files in a way that uses consistent and clear paths to avoid confusion when linking between directories.

Adding Tooltips

You can add a tooltip to your links using the title attribute. When users hover over the link, the tooltip will appear:

<a href="https://example.com" title="Click to visit Example">Visit Example</a>

Clickable Email Links

You can create links that allow users to send an email by using the mailto: prefix:

<a href="mailto:example@example.com">Email Us</a>

Clicking on this link will open the user’s default email client with a new message addressed to example@example.com.

Linking to Specific Sections

You can create links to specific sections of a page by using the id attribute and the # symbol. For example:

<a href="#section1">Go to Section 1</a>
<h2 id="section1">Section 1</h2>

When users click on the link, the browser will scroll to the element with the id="section1".

You can also link to a specific section on a different page by including the file name followed by the # symbol and the section's id. For example:

<a href="about-us.html#section2">Go to Section 2 on About Us</a>
<!-- In a different file -->
<h2 id="section2">Section 2</h2>

This link assumes that there is a file named about-us.html in the same directory, and it contains an element with id="section2". When clicked, the browser will navigate to that page and scroll directly to the specified section.

Best Practices for Links

  • Always use meaningful link text. Instead of "Click Here," use descriptive text like "Learn More About Us."
  • Use relative paths for internal links and absolute paths for external links.
  • Test your links to ensure they lead to the correct destination.
  • Use CSS to style your links for better visual appeal.

Accessibility and Links

Links should be accessible to everyone, including users relying on screen readers. Ensure that:

  • The link text describes its destination clearly.
  • The tabindex attribute is used, if necessary, to ensure proper tab navigation.
  • All important links are keyboard accessible.

HTML Links: Exercises

1. Create a Link to an External Website

Create an anchor tag that links to https://example.com. The link text should say "Visit Example". Make sure it opens in a new tab.

<a href="https://example.com" target="_blank">Visit Example</a>

2. Link to an Internal Page

Create a link that navigates to a page called about.html. The link text should say "Learn More About Us".

<a href="about.html">Learn More About Us</a>

3. Add a Tooltip to a Link

Create a link to https://example.com with the text "Hover Over Me". Add a tooltip that says "This is an example link".

<a href="https://example.com" title="This is an example link">Hover Over Me</a>

4. Create an Email Link

Create a link that allows users to send an email to info@stromayer.com. The link text should say "Contact Us".

<a href="mailto:info@stromayer.com">Contact Us</a>

5. Link to a Specific Section

Create a link that navigates to a section with the ID features on the same page. The link text should say "Go to Features".

<a href="#features">Go to Features</a>
<h2 id="features">Features</h2>
