JS Iterating Arrays

Iterating through an array allows us to process each element individually. JavaScript provides multiple ways to loop through arrays, making it easy to access and manipulate elements efficiently.

Common Ways to Iterate Over Arrays

Here are some common methods to loop through an array:

  • for loop – A basic way to iterate over an array using an index.
  • while loop – Iterates through an array based on a condition.
  • for...of loop – A simpler way to loop through array elements directly.

Using a for Loop

A for loop is commonly used to iterate through an array using an index.

let numbers = [10, 20, 30, 40];
for (let i = 0; i < numbers.length; i++) {
console.log(numbers[i]);
}

Using a while Loop

A while loop can also be used to iterate through an array by updating the index manually.

let fruits = ["Apple", "Banana", "Cherry"];
let i = 0;
while (i < fruits.length) {
console.log(fruits[i]);
i++;
}

Using the for...of Loop

The for...of loop allows us to loop through array elements directly without using an index.

let colors = ["Red", "Green", "Blue"];
for (let color of colors) {
console.log(color);
}

}
console.log("Jessica");
console.log("James");
console.log("Charlie");

3. Using a for...of Loop
