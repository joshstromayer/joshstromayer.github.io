JS Logical Operators

Logical operators in JavaScript are used to combine or modify boolean values (true or false). They are commonly used in conditions and decision-making statements.

Types of Logical Operators

JavaScript provides three main logical operators:

  • && (AND) – Returns true if both conditions are true.
  • || (OR) – Returns true if at least one condition is true.
  • ! (NOT) – Reverses the value of a boolean.

Using the && (AND) Operator

The && operator checks if both conditions are true. If any condition is false, the result is false.

console.log(5 > 3 && 10 > 8); // Output: true
console.log(5 > 3 && 10 < 8); // Output: false

Using the || (OR) Operator

The || operator checks if at least one condition is true. If all conditions are false, the result is false.

console.log(5 > 3 || 10 < 8); // Output: true
console.log(5 < 3 || 10 < 8); // Output: false

Using the ! (NOT) Operator

The ! operator reverses a boolean value. If a value is true, ! makes it false, and vice versa.

console.log(!true); // Output: false
console.log(!false); // Output: true

Combining Logical Operators

Logical operators can be combined to create complex conditions.

let age = 20;
console.log(age > 18 && age < 30); // Output: true
console.log(!(age > 18)); // Output: false
