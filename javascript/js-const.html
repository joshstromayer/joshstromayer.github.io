JS Const

In JavaScript, the const keyword is used to declare variables that cannot be reassigned after they are initialized. This makes const ideal for storing values that should remain unchanged throughout the program.

Declaring Constants

When using const, you must assign a value at the time of declaration. You cannot declare a const variable and then assign a value later.

Example:

const pi = 3.14;
console.log(pi); // Output: 3.14

pi = 3.14159; // Error: Assignment to constant variable

Since pi was declared with const, trying to change its value results in an error.

Constants Must Be Assigned a Value

Unlike let and var, a const variable must be assigned a value when it is declared. If you try to declare it without an initial value, you will get an error.

Example:

const country; // Error: Missing initializer in const declaration

Every const declaration must include an assignment at the time of creation.

Block Scope

Like let, const is block-scoped. This means it is only accessible within the block where it was defined.

Example:

if (true) {
  const message = "Hello, World!";
}

console.log(message); // Error: message is not defined

The variable message only exists inside the if block. Trying to access it outside the block causes an error.

When to Use const?

It is recommended to use const whenever you know the value of a variable should not change. Use let if the value needs to be updated later.

JS Const: Exercises

1. Declaring a Constant

Declare a constant named gravity and assign it the value 9.81. Then, try changing its value and observe what happens.

const gravity = 9.81;
console.log(gravity); // Output: 9.81

gravity = 10; // Error: Assignment to constant variable

2. Block Scope of const

Look at the following code snippet and predict what will happen:

if (true) {
  const city = "London";
}

console.log(city); // What happens here?
console.log(city); // Error: city is not defined (block-scoped)

3. Reassigning a const Variable

Consider the following code. What happens when you try to change the value of a const variable?

const temperature = 30;
console.log(temperature);

temperature = 35;
console.log(temperature);
const temperature = 30;
console.log(temperature); // Output: 30

temperature = 35; // Error: Assignment to constant variable

4. Using const Correctly

You need to declare a constant for the speed of light, which is 299792458 meters per second. Use const to ensure it does not change.

const speedOfLight = 299792458;
console.log(speedOfLight); // Output: 299792458
