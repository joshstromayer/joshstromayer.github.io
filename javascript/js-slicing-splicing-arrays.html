JS Slicing and Splicing Arrays

JavaScript provides methods to extract and modify parts of an array. The slice() and splice() methods are useful for working with subsets of an array.

Difference Between slice() and splice()

The slice() method extracts a section of an array without modifying the original array, while splice() modifies the original array by adding or removing elements.

Using slice()

The slice() method returns a portion of an array without altering the original array. It takes two arguments: the start index (inclusive) and the end index (exclusive).

let fruits = ["Apple", "Banana", "Cherry", "Date", "Elderberry"];
let slicedFruits = fruits.slice(1, 4);
console.log(slicedFruits); // Output: ["Banana", "Cherry", "Date"]
console.log(fruits); // Original array remains unchanged

Using splice()

The splice() method removes or adds elements to an array. It takes at least two arguments: the start index and the number of elements to remove. Optional additional arguments specify elements to insert.

let colors = ["Red", "Green", "Blue", "Yellow"];
colors.splice(2, 1); // Removes one element at index 2
console.log(colors); // Output: ["Red", "Green", "Yellow"]

Inserting Elements with splice()

The splice() method can also insert elements into an array.

let numbers = [10, 20, 40];
numbers.splice(2, 0, 30); // Inserts 30 at index 2
console.log(numbers); // Output: [10, 20, 30, 40]

JS Slicing and Splicing Arrays: Exercises

1. Using slice()

What will be the output of the following code?

let animals = ["Lion", "Tiger", "Elephant", "Giraffe"];
let selected = animals.slice(1, 3);
console.log(selected);
console.log(selected); // Output: ["Tiger", "Elephant"]

2. Using splice() to Remove Elements

What will be the output of the following code?

let fruits = ["Apple", "Banana", "Mango", "Cherry"];
fruits.splice(1, 2);
console.log(fruits);
console.log(fruits); // Output: ["Apple", "Cherry"]

3. Using splice() to Insert Elements

What will be the output of the following code?

let numbers = [5, 15, 25];
numbers.splice(1, 0, 10, 20);
console.log(numbers);
console.log(numbers); // Output: [5, 10, 20, 15, 25]

4. Mixing slice() and splice()

What will be the output of this code?

let letters = ["A", "B", "C", "D", "E"];
let part = letters.slice(1, 4);
part.splice(1, 1, "X");
console.log(part);
console.log(part); // Output: ["B", "X", "D"]
