JS Data Types

In JavaScript, a data type defines the kind of value that a variable can hold. JavaScript is a dynamically typed language, meaning you don't need to explicitly specify a type when declaring a variable.

Primitive Data Types

JavaScript has six primitive data types. These types represent single values and are immutable (cannot be changed).

  • String – Represents text (e.g., "Hello", 'JavaScript').
  • Number – Represents both integers and decimals (e.g., 42, 3.14).
  • Boolean – Represents either true or false.
  • Undefined – A variable that has been declared but not assigned a value.
  • Null – Represents an intentional absence of a value.
  • BigInt – Used for very large integers beyond the Number limit.

Example: Primitive Data Types

let name = "Emma"; // String
let age = 25; // Number
let isStudent = true; // Boolean
let score; // Undefined
let emptyValue = null; // Null
let largeNumber = 123456789012345678901234567890n; // BigInt

console.log(name, age, isStudent, score, emptyValue, largeNumber);

Special Type: Symbol

JavaScript also has a unique primitive type called Symbol, which is used to create unique values.

Example:

let id = Symbol("uniqueID");
console.log(id); // Output: Symbol(uniqueID)

Dynamic Typing

Since JavaScript is dynamically typed, you can assign different types of values to the same variable.

Example:

let variable = "Hello"; // String
console.log(variable);

variable = 100; // Now it's a Number
console.log(variable);

JS Data Types: Exercises

1. Identifying Data Types

Look at the following variables. What type of data does each one store?

let x = "JavaScript";
let y = 42;
let z = false;
let w;
let v = null;
x = "JavaScript"; // String
y = 42; // Number
z = false; // Boolean
w = undefined; // Undefined
v = null; // Null

2. Changing Data Types

Declare a variable score with a number value. Then, change it to a string and print both values.

let score = 85;
console.log(score); // Output: 85 (Number)

score = "Eighty-Five";
console.log(score); // Output: Eighty-Five (String)

3. Working with null and undefined

What is the difference between null and undefined? Write a small example showing both.

let notDefined;
console.log(notDefined); // Output: undefined

let empty = null;
console.log(empty); // Output: null

4. Using BigInt

Create a BigInt variable and print its value.

let bigNumber = 9007199254740991n;
console.log(bigNumber); // Output: 9007199254740991n
