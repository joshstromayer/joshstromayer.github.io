JS Variables

Variables in JavaScript allow you to store and manage data in your programs. They act as containers for values that can be used and manipulated throughout your code.

Declaring Variables

JavaScript provides three keywords for declaring variables: var, let, and const. Each has different rules and use cases.

Using var

The var keyword was traditionally used to declare variables in JavaScript. However, it has some quirks related to scoping, so it is now less commonly used.

var name = "Stevie"; // Declaring a variable
console.log(name); // Output: Stevie

Using let

The let keyword is now the preferred way to declare variables in JavaScript. It allows you to create variables that can be reassigned later.

let age = 25;
age = 26; // Reassigning a new value
console.log(age); // Output: 26

Using const

The const keyword is used for variables that should not change once they are assigned.

const country = "USA";
console.log(country); // Output: USA

Rules for Naming Variables

  • Variable names must start with a letter, _ (underscore), or $ (dollar sign).
  • They cannot start with a number.
  • JavaScript is case-sensitive: age and Age are different variables.
  • Variable names should be descriptive (e.g., userName instead of x).

JS Variables: Exercises

1. Declare a Variable

Declare a variable named favoriteColor and assign it the value "blue". Then, log it to the console.

let favoriteColor = "blue";
console.log(favoriteColor);

2. Reassign a Variable

Declare a variable named userAge using let, assign it a value of 30, and then change it to 31.

let userAge = 30;
userAge = 31;
console.log(userAge); // Output: 31

3. Create a Constant

Declare a constant variable named PI with the value 3.14159 and log it to the console.

const PI = 3.14159;
console.log(PI);

4. Incorrect Variable Naming

Which of the following variable names are invalid in JavaScript?

  • 1name
  • user-name
  • _private
  • $price

Hint: Numbers cannot be at the start, and hyphens are not allowed in variable names.
