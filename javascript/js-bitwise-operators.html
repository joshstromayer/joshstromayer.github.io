JS Bitwise Operators

Bitwise operators in JavaScript work on the individual bits of numbers. These operators are primarily used for low-level programming tasks, such as working with binary data, encryption, and optimizing calculations.

Understanding Binary Representation

Before diving into bitwise operations, it's essential to understand that numbers in JavaScript are stored in **binary format** (base-2). For example:

  • 5 in binary: 00000000000000000000000000000101
  • 3 in binary: 00000000000000000000000000000011

Types of Bitwise Operators

JavaScript provides six bitwise operators:

  • & (AND) – Sets bits to 1 only if both bits are 1.
  • | (OR) – Sets bits to 1 if either bit is 1.
  • ^ (XOR) – Sets bits to 1 if only one bit is 1.
  • ~ (NOT) – Inverts all bits.
  • << (Left Shift) – Shifts bits to the left.
  • >> (Right Shift) – Shifts bits to the right.

Bitwise AND (&)

The AND operator compares corresponding bits of two numbers. The result is **1** if both bits are **1**, otherwise, it is **0**.

console.log(5 & 3);
// 5 = 00000000000000000000000000000101
// 3 = 00000000000000000000000000000011
// -----------------------------
// Result = 00000000000000000000000000000001 (1)

Bitwise OR (|)

The OR operator compares corresponding bits of two numbers. The result is **1** if either bit is **1**.

console.log(5 | 3);
// 5 = 00000000000000000000000000000101
// 3 = 00000000000000000000000000000011
// -----------------------------
// Result = 00000000000000000000000000000111 (7)

Bitwise XOR (^)

The XOR operator returns **1** if only one of the bits is **1**, otherwise, it returns **0**.

console.log(5 ^ 3);
// 5 = 00000000000000000000000000000101
// 3 = 00000000000000000000000000000011
// -----------------------------
// Result = 00000000000000000000000000000110 (6)

Bitwise NOT (~)

The NOT operator inverts the bits of a number.

console.log(~5);
// 5 = 00000000000000000000000000000101
// NOT = 11111111111111111111111111111010 (-6)

Bitwise Shift Operators

JavaScript supports two shift operators:

  • << (Left Shift) – Shifts bits to the left, filling with zeros.
  • >> (Right Shift) – Shifts bits to the right, maintaining the sign.
console.log(5 << 1); // Output: 10 (Shifts left by 1 bit)
console.log(5 >> 1); // Output: 2 (Shifts right by 1 bit)
