JS Inheritance

In JavaScript, inheritance allows one class to acquire properties and methods from another class. This helps in reusing code and maintaining an organized structure.

Extending a Class

The extends keyword is used to create a class that inherits from another class.

class Animal {
  constructor(name) {
    this.name = name;
  }
  speak() {
    console.log(`${this.name} makes a noise.`);
  }
}

class Dog extends Animal {
  bark() {
    console.log(`${this.name} barks!`);
  }
}

let dog1 = new Dog("Buddy");
dog1.speak(); // Output: Buddy makes a noise.
dog1.bark(); // Output: Buddy barks!

Using super() in Inheritance

The super() function allows us to call the constructor of the parent class.

class Animal {
  constructor(name) {
    this.name = name;
  }
}

class Dog extends Animal {
  constructor(name, breed) {
    super(name);
    this.breed = breed;
  }
}

let dog2 = new Dog("Rocky", "Golden Retriever");
console.log(dog2.name); // Output: Rocky
console.log(dog2.breed); // Output: Golden Retriever

Overriding Methods

A subclass can override a method from its parent class by defining a method with the same name.

class Animal {
  speak() {
    console.log("An animal makes a sound.");
  }
}

class Cat extends Animal {
  speak() {
    console.log("Meow!");
  }
}

let cat1 = new Cat();
cat1.speak(); // Output: Meow!

1. Understanding Inheritance

What will be the output of the following code?

class Parent {
  sayHello() {
    console.log("Hello from Parent!");
  }
}

class Child extends Parent {}

let child1 = new Child();
child1.sayHello();
Hello from Parent!

2. Calling super()

What will be the output of the following code?

class Vehicle {
  constructor(type) {
    this.type = type;
  }
}

class Car extends Vehicle {
  constructor(type, brand) {
    super(type);
    this.brand = brand;
  }
}

let myCar = new Car("Sedan", "Honda");
console.log(myCar.type, myCar.brand);
Sedan Honda
