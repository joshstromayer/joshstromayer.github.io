JS Output

JavaScript provides multiple ways to display output on a web page. Whether you want to show text on the screen, display messages in pop-ups, or log information in the console, JavaScript has built-in methods to handle output.

Using console.log()

The console.log() method prints messages to the browser's console. This is useful for debugging and testing JavaScript code.

  console.log("Hello, World!"); // Output appears in the browser console

To view the output, open the browser's developer tools (press F12 or Ctrl + Shift + I on most browsers) and go to the Console tab.

Using document.write()

The document.write() method writes directly to the HTML page. It is mostly used for testing because it overwrites the existing content of the document.

  document.write("This text is written to the page!");

Warning: Using document.write() after the page has loaded will erase all existing content.

Using alert()

The alert() method displays a pop-up message in the browser. It is useful for simple notifications or warnings.

  alert("This is an alert message!");

When executed, this code shows a dialog box with an OK button. The user must click OK to continue.

Using innerHTML

The innerHTML property allows you to change the content of an HTML element dynamically.

<!-- HTML -->
<p id="demo">Original Text</p>
<button onclick="changeText()">Click Me</button>

<!-- JavaScript -->
function changeText() {
  document.getElementById("demo").innerHTML = "Text changed!";
}

Clicking the button modifies the paragraph's content.

Using window.prompt()

The prompt() method allows users to input a value in a dialog box.

  let userName = prompt("Enter your name:");
  alert("Hello, " + userName + "!");

The user enters their name, and JavaScript displays a greeting with the inputted name.

Using window.confirm()

The confirm() method displays a dialog box with OK and Cancel buttons, allowing users to make a choice.

  let response = confirm("Do you want to proceed?");
  if (response) {
    alert("You clicked OK!");
  } else {
    alert("You clicked Cancel!");
  }

The confirm dialog helps get user confirmation before performing an action.
