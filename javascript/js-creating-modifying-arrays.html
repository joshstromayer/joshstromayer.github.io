JS Creating and Modifying Arrays

In JavaScript, arrays can be created in different ways. The most common way is using array literals, but arrays can also be created using the new Array() constructor.

Creating an Array Using `[ ]`

The simplest and most commonly used way to create an array is with square brackets `[ ]`.

Example: Creating an array using `[ ]`

let numbers = [10, 20, 30, 40];
console.log(numbers); // Output: [10, 20, 30, 40]

This method is preferred because it is shorter and more readable.

Creating an Array Using new Array()

Another way to create an array is by using the new Array() constructor. However, this method is not recommended unless you need to create an array of a specific length.

Example: Creating an array using new Array()

let colors = new Array("Red", "Green", "Blue");
console.log(colors); // Output: ["Red", "Green", "Blue"]

This method works similarly to `[ ]`, but using new Array(5) creates an empty array with five slots.

Modifying Array Elements

You can modify an array by changing its values using their index number.

Example: Changing an array element

let fruits = ["Apple", "Banana", "Cherry"];
fruits[1] = "Mango"; // Change "Banana" to "Mango"
console.log(fruits); // Output: ["Apple", "Mango", "Cherry"]

Adding Elements to an Array

You can add new elements to an array by assigning a value to a new index.

Example: Adding a new element at index 3

let animals = ["Dog", "Cat", "Elephant"];
animals[3] = "Tiger";
console.log(animals); // Output: ["Dog", "Cat", "Elephant", "Tiger"]

However, this method is not the best approach for adding elements. There are **better ways** to add and remove elements, which will be covered in the next lessons.

Conclusion

JavaScript arrays can be created using `[ ]` or new Array(), but using `[ ]` is the recommended approach. You can also modify elements by accessing them using their index.
