JS Creating Objects

In JavaScript, there are multiple ways to create objects. The most common methods are using object literals {} and the new Object() constructor.

Creating an Object Using an Object Literal

The easiest and most common way to create an object is by using curly braces {}, also known as an object literal.

let person = {
name: "Jessica",
age: 25,
city: "Paris"
};
console.log(person);

Creating an Object Using new Object()

Another way to create an object is by using the new Object() constructor. This method is less commonly used but still valid.

let car = new Object();
car.brand = "Ford";
car.model = "Mustang";
car.year = 2023;
console.log(car);

Comparison of Object Creation Methods

Both methods achieve the same result, but using object literals {} is generally preferred because it is shorter and more readable.

4. Object Comparison

Which of the following is the preferred way to create an object in JavaScript?

let obj1 = {};

The preferred way is let obj1 = {}; because it is shorter and more readable.
