For Loops

In JavaScript, a for loop is used to execute a block of code multiple times. It is useful when you want to repeat an action a specific number of times.

Basic Syntax

A for loop consists of three parts:

  • Initialization - Declares a variable to start the loop.
  • Condition - Defines when the loop should stop.
  • Iteration - Updates the loop variable after each cycle.

Example:

for (let i = 1; i <= 5; i++) {
    console.log("Iteration number:", i);
}

This loop starts with i = 1, runs while i <= 5, and increases i by 1 each time.

Looping Through an Array

You can use a for loop to go through an array and access its elements.

Example: Looping through an array of colors

let colors = ["Red", "Green", "Blue"];
for (let i = 0; i < colors.length; i++) {
    console.log(colors[i]);
}

Skipping Iterations with continue

The continue statement allows you to skip an iteration without stopping the entire loop.

Example: Skipping even numbers

for (let i = 1; i <= 5; i++) {
    if (i % 2 === 0) continue;
    console.log(i);
}

Exiting a Loop Early with break

The break statement stops a loop before it completes all its iterations.

Example: Stopping at 3

for (let i = 1; i <= 5; i++) {
    if (i === 3) break;
    console.log(i);
}

When to Use For Loops?

- Use for loops when you know how many times an action should repeat.
- Use them for iterating over arrays, sequences, or performing repeated calculations.

}

3. Using continue

for (let i = 1; i <= 10; i++) {
    console.log(i);
}
for (let i = 1; i <= 10; i++) {
    if (i % 3 === 0) continue;
    console.log(i);
}

for (let i = 1; i <= 10; i++) {
    console.log(i);
}

    if (i === 4) break;
    console.log(i);
