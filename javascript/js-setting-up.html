JS Setup

Before writing JavaScript, you need to know where and how to include it in your web pages. JavaScript can be added directly inside an HTML document or stored in an external file.

Using JavaScript Inside HTML

JavaScript can be placed inside an HTML file using the <script> tag. It can be written in two ways:

  • Inline JavaScript: Inside an HTML element.
  • Internal JavaScript: Inside a <script> tag.

Example of Inline JavaScript:

<button onclick="alert('Hello, World!')">Click Me</button>

In this example, JavaScript is directly inside the HTML button element. When the button is clicked, an alert box appears.

Example of Internal JavaScript:

<!DOCTYPE html>
<html>
  <head>
    <title>JavaScript Example</title>
  </head>
  <body>

    <button onclick="showMessage()">Click Me</button>
    <p id="output"></p>

    <script>
    function showMessage() {
      document.getElementById("output").innerHTML = "Hello from JavaScript!";
    }
    </script>

  </body>
</html>

Here, JavaScript is placed inside the <script> tag in the HTML document.

External JavaScript

Instead of writing JavaScript inside an HTML file, you can store it in an external file with a .js extension and link it using the src attribute inside a <script> tag.

Example of External JavaScript:

<!DOCTYPE html>
<html>
  <head>
    <title>JavaScript Example</title>
    <script src="script.js"></script>
  </head>
  <body>

    <button onclick="showMessage()">Click Me</button>
    <p id="output"></p>

  </body>
</html>

External JavaScript File (script.js):

function showMessage() {
  document.getElementById("output").innerHTML = "Hello from an external file!";
}

Here, the JavaScript code is written in a separate file named script.js. The HTML file links to it using the <script src="script.js"> tag.

Where to Place JavaScript in HTML?

JavaScript can be placed in two main locations in an HTML document:

  • Inside the <head> tag: The script runs before the page loads. This can cause issues if it tries to manipulate elements that havenâ€™t loaded yet.
  • At the end of the <body> tag: Recommended, because it allows the HTML to load first before JavaScript runs.

Example of JavaScript at the bottom of the page:

<body>
  <h1>My Web Page</h1>

  <script src="script.js"></script>
</body>

Best Practices

When setting up JavaScript in your project, follow these best practices:

  • Use external JavaScript files for better organization and reusability.
  • Place JavaScript at the end of the <body> tag to avoid delaying page rendering.
  • Always test your JavaScript in different browsers to ensure compatibility.
