JS Comparison Operators

Comparison operators in JavaScript are used to compare values. They return a boolean value: true or false. These operators are commonly used in decision-making structures, such as if statements.

Types of Comparison Operators

JavaScript provides several comparison operators:

  • == (Equal to) – Checks if two values are equal, but does not compare data types.
  • === (Strict equal to) – Checks if two values are equal and also of the same data type.
  • != (Not equal to) – Checks if two values are not equal.
  • !== (Strict not equal to) – Checks if two values are not equal or not of the same data type.
  • > (Greater than) – Checks if the left value is greater than the right value.
  • < (Less than) – Checks if the left value is smaller than the right value.
  • >= (Greater than or equal to) – Checks if the left value is greater than or equal to the right value.
  • <= (Less than or equal to) – Checks if the left value is smaller than or equal to the right value.

Using the == vs. === Operators

The == operator checks only the value, while the === operator checks both the value and the data type.

console.log(5 == "5"); // Output: true (Values are the same, but different types)
console.log(5 === "5"); // Output: false (Values are the same, but types are different)

Using != vs. !==

The != operator checks if two values are different, while !== also checks if their data types are different.

console.log(10 != "10"); // Output: false (Values are the same)
console.log(10 !== "10"); // Output: true (Different data types)

Greater and Less Than Operators

The > and < operators check if one value is greater or smaller than another.

console.log(8 > 5); // Output: true
console.log(3 < 7); // Output: true

JS Comparison Operators: Exercises

1. Checking Equality

What will be the output of the following code?

console.log(15 == "15");
console.log(15 === "15");
console.log(15 == "15"); // Output: true
console.log(15 === "15"); // Output: false

2. Comparing Numbers

Predict the output of the following comparison:

console.log(20 > 10);
console.log(5 <= 5);
console.log(20 > 10); // Output: true
console.log(5 <= 5); // Output: true

3. Using != and !==

What will be the output of the following?

console.log("10" != 10);
console.log("10" !== 10);
console.log("10" != 10); // Output: false
console.log("10" !== 10); // Output: true

4. Complex Comparison

What will be the output of this code?

let x = 5;
console.log(x > 3 && x < 10);
console.log(x < 3 || x > 10);
console.log(x > 3 && x < 10); // Output: true
console.log(x < 3 || x > 10); // Output: false
