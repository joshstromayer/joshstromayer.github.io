JS Syntax

Every programming language has a set of rules that define how code should be written. JavaScript follows a structured syntax that allows developers to write clean, readable, and functional code.

JavaScript Statements

JavaScript code consists of statements, which are instructions the browser executes. Each statement typically ends with a semicolon though it's not always required.

  console.log("Hello, World!"); // This is a JavaScript statement

JavaScript statements are executed one by one, in order from top to bottom.

Case Sensitivity

JavaScript is case-sensitive, meaning variables, functions, and keywords must be written with consistent capitalization.

  let myVariable = 10;
  console.log(MyVariable); // ❌ Error: MyVariable is not defined

In this example, myVariable and MyVariable are treated as different identifiers.

JavaScript Keywords

JavaScript has reserved keywords that have special meanings. You cannot use these words as variable or function names.

  • var – Declares a variable.
  • let – Declares a block-scoped variable.
  • const – Declares a constant (unchangeable) variable.
  • function – Defines a function.
  • return – Returns a value from a function.
  • if, else, switch – Used for conditional statements.
  • for, while – Used for loops.

Comments in JavaScript

Comments are ignored by JavaScript but are useful for explaining code. JavaScript supports:

  • Single-line comments using //.
  • Multi-line comments using /* ... */.
  // This is a single-line comment
  let x = 10; // Variable declaration

  /*
  This is a multi-line comment.
  It spans multiple lines.
  */

JavaScript Variables

Variables store data that can be used and modified in JavaScript. You can declare variables using:

  • var – The old way of declaring variables (not recommended).
  • let – The modern way to declare a variable that can be updated.
  • const – Used for values that should not be changed.
  var oldVar = "I am outdated";
  let modernVar = "I can be updated";
  const constantVar = "I cannot be changed";

Data Types in JavaScript

JavaScript supports different data types, including:

  • String – Text values, written inside quotes ("Hello" or 'World').
  • Number – Numeric values (42, 3.14).
  • Boolean – True or false values (true, false).
  • Array – A collection of values ([1, 2, 3]).
  • Object – A collection of key-value pairs ({ name: "Stevie", age: 30 }).

JavaScript Functions

A function is a block of code that runs when called. Functions allow you to reuse code.

  function greet() {
    console.log("Hello, world!");
  }

  greet(); // Calls the function

JavaScript Operators

JavaScript provides different types of operators to perform operations on values.

  • Arithmetic Operators: +, -, *, /, %
  • Comparison Operators: ==, ===, !=, >, <
  • Logical Operators: && (AND), || (OR), ! (NOT)
