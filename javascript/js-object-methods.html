JS Object Methods

In JavaScript, an object method is a function that is stored as a property inside an object. Methods allow objects to perform actions and work with their own properties.

Defining a Method in an Object

A method is defined by assigning a function to an object property.

let person = {
name: "Sophia",
age: 27,
greet: function() {
console.log("Hello, my name is " + this.name);
}
};
person.greet(); // Output: "Hello, my name is Sophia"

Using the this Keyword

Inside an object method, the this keyword refers to the object itself. This allows the method to access other properties of the same object.

let car = {
brand: "Toyota",
model: "Corolla",
getInfo: function() {
return this.brand + " " + this.model;
}
};
console.log(car.getInfo()); // Output: "Toyota Corolla"

Adding Methods to an Existing Object

We can also add methods to an existing object dynamically.

let animal = {
type: "Dog",
breed: "Labrador"
};
animal.speak = function() {
return "Woof!";
};
console.log(animal.speak()); // Output: "Woof!"

JS Object Methods: Exercises

1. Calling a Method

What will be the output of the following code?

let user = {
name: "Liam",
sayHello: function() {
return "Hello " + this.name;
}
};
console.log(user.sayHello());
console.log(user.sayHello()); // Output: "Hello Liam"

2. Using this in an Object

What will be the output of the following code?

let laptop = {
brand: "HP",
getBrand: function() {
return this.brand;
}
};
console.log(laptop.getBrand());
console.log(laptop.getBrand()); // Output: "HP"

3. Adding a New Method

What will be the output of the following code?

let fruit = {
name: "Banana"
};
fruit.getName = function() {
return this.name;
};
console.log(fruit.getName());
console.log(fruit.getName()); // Output: "Banana"

4. Missing this Keyword

What will be the output of this code?

let city = {
name: "New York",
getCity: function() {
return name;
}
};
console.log(city.getCity());
// ReferenceError: name is not defined (because "this.name" should be used instead)
