Do While Loops

A do while loop in JavaScript is similar to a while loop, but it guarantees that the code inside the loop will execute at least once, even if the condition is false.

Basic Syntax

The do while loop first executes the code block and then checks the condition. If the condition is true, the loop runs again; otherwise, it stops.

Example: Printing numbers from 1 to 5

let i = 1;
do {
    console.log("Number:", i);
    i++;
} while (i <= 5);

The loop starts with i = 1, prints the number, then checks if i <= 5. If true, it repeats the process.

Ensuring Execution at Least Once

The key difference between a while loop and a do while loop is that a do while loop always executes at least once, even if the condition is initially false.

Example: A loop that runs once even if the condition is false

let count = 10;
do {
    console.log("This runs at least once!");
    count++;
} while (count < 5);

Since count starts at 10 (greater than 5), the condition is already false, but the message still prints once before stopping.

Using break in a Do While Loop

The break statement can be used inside a do while loop to exit early.

Example: Stopping a loop when a number reaches 3

let num = 1;
do {
    if (num === 3) break;
    console.log(num);
    num++;
} while (num <= 5);

Using continue in a Do While Loop

The continue statement skips the rest of the loop iteration and moves to the next cycle.

Example: Skipping even numbers

let x = 0;
do {
    x++;
    if (x % 2 === 0) continue;
    console.log(x);
} while (x < 5);

When to Use Do While Loops?

- Use do while loops when you need to ensure the loop runs at least once.
- Useful when gathering user input or working with conditions that might initially be false.
