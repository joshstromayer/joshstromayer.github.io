JS Arithmetic Operators

JavaScript provides several arithmetic operators that allow you to perform mathematical calculations. These operators work with numbers and return numeric results.

Basic Arithmetic Operators

The most commonly used arithmetic operators in JavaScript include:

  • + (Addition) – Adds two numbers.
  • - (Subtraction) – Subtracts one number from another.
  • * (Multiplication) – Multiplies two numbers.
  • / (Division) – Divides one number by another.
  • % (Modulus) – Returns the remainder of a division.
  • ** (Exponentiation) – Raises a number to a power.

Example: Using Arithmetic Operators

let a = 10;
let b = 3;

console.log(a + b); // Output: 13 (Addition)
console.log(a - b); // Output: 7 (Subtraction)
console.log(a * b); // Output: 30 (Multiplication)
console.log(a / b); // Output: 3.333... (Division)
console.log(a % b); // Output: 1 (Modulus)
console.log(a ** b); // Output: 1000 (Exponentiation)

Increment and Decrement Operators

JavaScript also provides operators for increasing or decreasing a value by 1.

  • ++ (Increment) – Increases a value by 1.
  • -- (Decrement) – Decreases a value by 1.

Example: Increment and Decrement

let count = 5;
count++; // Increases count by 1
console.log(count); // Output: 6

count--; // Decreases count by 1
console.log(count); // Output: 5

Order of Operations (Operator Precedence)

JavaScript follows the standard mathematical rules for order of operations:

  • Parentheses () have the highest precedence.
  • Exponentiation ** comes next.
  • Multiplication *, Division /, and Modulus % follow.
  • Addition + and Subtraction - have the lowest precedence.

Example: Operator Precedence

let result = 10 + 5 * 2;
console.log(result); // Output: 20 (Multiplication happens first)

let correctResult = (10 + 5) * 2;
console.log(correctResult); // Output: 30 (Parentheses change precedence)

JS Arithmetic Operators: Exercises

1. Basic Arithmetic Operations

console.log(15 + 5);
console.log(20 - 7);
console.log(6 * 3);
console.log(25 / 5);
console.log(15 + 5); // Output: 20
console.log(20 - 7); // Output: 13
console.log(6 * 3); // Output: 18
console.log(25 / 5); // Output: 5

2. Modulus and Exponentiation

What will be the output of the following operations?

console.log(10 % 3);
console.log(2 ** 4);
console.log(10 % 3); // Output: 1 (Remainder of 10 ÷ 3)
console.log(2 ** 4); // Output: 16 (2 raised to the power of 4)

3. Increment and Decrement

Predict the output of the following code:

let num = 8;
console.log(num++);
console.log(num);
console.log(num++); // Output: 8 (Post-increment returns the value before incrementing)
console.log(num); // Output: 9

4. Operator Precedence

Predict the result of the following expression:

let result = 10 + 2 * 5 - 3;
console.log(result);
let result = 10 + 2 * 5 - 3;
console.log(result); // Output: 17
// Multiplication (2 * 5 = 10), then Addition (10 + 10 = 20), then Subtraction (20 - 3 = 17)
