While Loops

A while loop in JavaScript executes a block of code as long as a specified condition is true. Unlike a for loop, a while loop is useful when the number of iterations is unknown beforehand.

Basic Syntax

The while loop evaluates a condition before each iteration. If the condition is true, the loop body executes.

Example: Printing numbers from 1 to 5

let i = 1;
while (i <= 5) {
    console.log("Number:", i);
    i++;
}

The loop starts with i = 1, and continues running while i <= 5. The value of i increases by 1 in each iteration.

Looping Until a Condition is Met

The while loop is ideal when you donâ€™t know in advance how many times the loop should run.

Example: Rolling a dice until a 6 is rolled

let diceRoll;
while (diceRoll !== 6) {
    diceRoll = Math.floor(Math.random() * 6) + 1;
    console.log("Rolled:", diceRoll);
}

Using break in a While Loop

The break statement can be used to exit a while loop when a specific condition is met.

Example: Stopping a loop when a number reaches 3

let num = 1;
while (num <= 5) {
    if (num === 3) break;
    console.log(num);
    num++;
}

Using continue in a While Loop

The continue statement allows skipping an iteration without stopping the entire loop.

Example: Skipping even numbers

let x = 0;
while (x < 5) {
    x++;
    if (x % 2 === 0) continue;
    console.log(x);
}

When to Use While Loops?

- Use while loops when the number of iterations is unknown in advance.
- Ideal for cases where the loop should run until a condition changes dynamically.
