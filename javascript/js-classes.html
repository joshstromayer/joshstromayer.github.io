JS Classes

In JavaScript, classes provide a structured way to create objects. They help organize code and make object creation easier.

Defining a Class

A class is defined using the class keyword. The constructor method inside a class initializes the object's properties.

class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }
}

let person1 = new Person("Liam", 25);
console.log(person1.name); // Output: Liam

Adding Methods to a Class

You can add methods to a class to define actions that objects of that class can perform.

class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }
  greet() {
    console.log(`Hello, my name is ${this.name}.`);
  }
}

let person2 = new Person("Emily", 30);
person2.greet(); // Output: Hello, my name is Emily.

Using Class Instances

When you create an object from a class, it is called an instance. Each instance has its own unique values.

let person3 = new Person("Nathan", 28);
let person4 = new Person("Sophia", 22);
console.log(person3.name); // Output: Nathan
console.log(person4.name); // Output: Sophia

JS Classes: Exercises

1. Creating a Class

What will be the output of the following code?

class Car {
  constructor(brand) {
    this.brand = brand;
  }
}

let myCar = new Car("Toyota");
console.log(myCar.brand);
console.log(myCar.brand); // Output: Toyota

2. Using a Method

What will be the output of the following code?

class Animal {
  constructor(name) {
    this.name = name;
  }
  speak() {
    console.log(`${this.name} makes a sound.`);
  }
}

let cat = new Animal("Whiskers");
cat.speak();
cat.speak(); // Output: Whiskers makes a sound.

3. Multiple Instances

What will be the output of the following code?

class Book {
  constructor(title, author) {
    this.title = title;
    this.author = author;
  }
}

let book1 = new Book("1984", "George Orwell");
let book2 = new Book("Brave New World", "Aldous Huxley");
console.log(book2.title);
console.log(book2.title); // Output: Brave New World
