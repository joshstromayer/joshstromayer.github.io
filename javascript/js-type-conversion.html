JS Type Conversion

JavaScript allows us to convert data from one type to another. This is called type conversion and can happen either automatically (implicit conversion) or manually (explicit conversion).

Implicit Type Conversion (Type Coercion)

JavaScript automatically converts one data type into another when needed. This is called type coercion.

Example: Number + String

let num = 10;
let str = "5";
let result = num + str;
console.log(result); // Output: "105" (Number is converted to a String)

In this example, JavaScript converts the number 10 into a string and concatenates it with "5".

Example: Boolean to Number

console.log(true + 5); // Output: 6 (true is converted to 1)
console.log(false + 10); // Output: 10 (false is converted to 0)

Explicit Type Conversion

We can manually convert a data type using JavaScript built-in methods like:

  • Number() – Converts a value to a number.
  • String() – Converts a value to a string.
  • Boolean() – Converts a value to a boolean.
  • parseInt() – Converts a string to an integer.
  • parseFloat() – Converts a string to a floating-point number.

Example: Converting Strings to Numbers

let str = "20";
let num = Number(str);
console.log(num); // Output: 20 (Converted to Number)

Example: Converting Numbers to Strings

let num = 15;
let str = String(num);
console.log(str); // Output: "15" (Converted to String)

Example: Converting Values to Boolean

console.log(Boolean(1)); // Output: true
console.log(Boolean(0)); // Output: false
console.log(Boolean("Hello")); // Output: true
console.log(Boolean("")); // Output: false

JS Type Conversion: Exercises

1. Implicit Type Conversion

Predict the output of the following code:

console.log("5" + 10);
console.log(10 + "5");
console.log("5" * 2);
console.log("10" - 3);
console.log("5" + 10); // Output: "510" (Number converted to string)
console.log(10 + "5"); // Output: "105" (Number converted to string)
console.log("5" * 2); // Output: 10 (String converted to number)
console.log("10" - 3); // Output: 7 (String converted to number)

2. Convert a String to a Number

Convert the string "42" into a number using JavaScript, then print the result.

let str = "42";
let num = Number(str);
console.log(num); // Output: 42

3. Convert a Boolean to a Number

Use JavaScript to convert true and false to numbers and print the results.

console.log(Number(true)); // Output: 1
console.log(Number(false)); // Output: 0

4. Convert a Number to a String

Convert the number 99 into a string and print the result.

let num = 99;
let str = String(num);
console.log(str); // Output: "99"
