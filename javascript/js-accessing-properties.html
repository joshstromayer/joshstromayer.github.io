JS Accessing Object Properties

After creating an object, we need to access its properties to retrieve values. JavaScript provides two ways to access object properties: **dot notation** and **bracket notation**.

Accessing Properties with Dot Notation

The most common way to access object properties is using **dot notation** (.).

let person = {
name: "Emma",
age: 28,
city: "London"
};
console.log(person.name); // Output: "Emma"

Accessing Properties with Bracket Notation

We can also access object properties using **bracket notation** ([]). This is useful when property names contain spaces or special characters.

let car = {
brand: "Tesla",
model: "Model S",
"top speed": 200
};
console.log(car["top speed"]); // Output: 200

Using Variables to Access Properties

With bracket notation, we can use variables to dynamically access object properties.

let book = {
title: "1984",
author: "George Orwell"
};
let key = "title";
console.log(book[key]); // Output: "1984"
