JS Introduction to Objects

Objects are one of the most important features in JavaScript. They allow us to store multiple values in a structured way using key-value pairs.

What is an Object?

An object in JavaScript is a collection of properties, where each property consists of a key (also known as a property name) and a corresponding value.

let person = {
name: "Stevie",
age: 30,
city: "New York"
};
console.log(person);

Why Use Objects?

Objects help organize data efficiently and allow us to store related information in a single entity. They are commonly used to represent real-world entities, such as a person, a car, or a user profile.

Basic Structure of an Object

An object consists of key-value pairs. The keys are always strings, and the values can be any data type, including numbers, strings, arrays, or even other objects.

let car = {
brand: "Toyota",
model: "Corolla",
year: 2022
};
console.log(car.model); // Output: "Corolla"
