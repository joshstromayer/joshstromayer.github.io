JS Introduction to Arrays

In JavaScript, an array is a special type of object that allows you to store multiple values in a single variable. Arrays are useful when working with lists of data, such as a collection of numbers, names, or even objects.

What is an Array?

An array is an ordered collection of values, where each value is stored at a specific index. Unlike regular variables that hold only a single value, an array can hold multiple values.

Example: Declaring an array with different data types

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits); // Output: ["Apple", "Banana", "Cherry"]

In this example, fruits is an array that contains three string values: "Apple", "Banana", and "Cherry".

Array Indexing

Each element in an array is assigned a numeric index, starting from 0. This means that the first item is at index `0`, the second at index `1`, and so on.

Example: Accessing elements by index

let colors = ["Red", "Green", "Blue"];
console.log(colors[0]); // Output: "Red"
console.log(colors[1]); // Output: "Green"
console.log(colors[2]); // Output: "Blue"

Why Use Arrays?

Arrays allow us to store multiple values in a structured way, making it easier to work with related data. Without arrays, we would have to store each value in separate variables.

Example: Without an array vs with an array

// Without an array
let name1 = "Jessica";
let name2 = "James";
let name3 = "Charlie";

// Using an array
let names = ["Jessica", "James", "Charlie"];

Using an array makes it easier to manage and manipulate the data, especially when dealing with large collections.

Arrays Can Store Different Data Types

Arrays in JavaScript can store different types of values, including numbers, strings, and even other arrays or objects.

Example: Mixed data types in an array

let mixedArray = ["Hello", 42, true];
console.log(mixedArray); // Output: ["Hello", 42, true]

Conclusion

Arrays are a powerful way to store and manage multiple values in JavaScript. In the next lesson, we will learn how to create and modify arrays.
