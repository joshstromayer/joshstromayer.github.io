JS Basic Array Methods

JavaScript provides several built-in methods to manipulate arrays efficiently. These methods allow us to add, remove, and modify elements within an array.

Common Array Methods

Here are some of the most commonly used array methods in JavaScript:

  • push() – Adds an element to the end of an array.
  • pop() – Removes the last element from an array.
  • shift() – Removes the first element from an array.
  • unshift() – Adds an element to the beginning of an array.
  • length – Returns the number of elements in an array.

Adding Elements: push() and unshift()

The push() method adds one or more elements to the end of an array, while the unshift() method adds elements to the beginning.

let fruits = ["Apple", "Banana"];
fruits.push("Orange");
console.log(fruits); // Output: ["Apple", "Banana", "Orange"]

fruits.unshift("Mango");
console.log(fruits); // Output: ["Mango", "Apple", "Banana", "Orange"]

Removing Elements: pop() and shift()

The pop() method removes the last element from an array, while shift() removes the first element.

let colors = ["Red", "Green", "Blue"];
colors.pop();
console.log(colors); // Output: ["Red", "Green"]

colors.shift();
console.log(colors); // Output: ["Green"]

Finding the Array Length

The length property returns the number of elements in an array.

let numbers = [10, 20, 30, 40];
console.log(numbers.length); // Output: 4
