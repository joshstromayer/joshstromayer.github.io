JS Modifying Object Properties

Once an object is created, its properties can be modified by assigning new values to them. JavaScript allows us to update object properties using **dot notation** or **bracket notation**.

Modifying Properties with Dot Notation

The most common way to modify an object property is using **dot notation** (.).

let person = {
name: "Emma",
age: 28,
city: "London"
};
person.age = 30;
console.log(person.age); // Output: 30

Modifying Properties with Bracket Notation

We can also modify object properties using **bracket notation** ([]).

let car = {
brand: "Tesla",
model: "Model S",
year: 2022
};
car["year"] = 2023;
console.log(car.year); // Output: 2023

Adding New Properties

JavaScript allows us to add new properties to an existing object simply by assigning a value to a new key.

let book = {
title: "1984",
author: "George Orwell"
};
book.pages = 328;
console.log(book.pages); // Output: 328

Deleting Object Properties

We can remove a property from an object using the delete keyword.

let laptop = {
brand: "Dell",
memory: "16GB"
};
delete laptop.memory;
console.log(laptop.memory); // Output: undefined
