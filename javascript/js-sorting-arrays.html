JS Sorting Arrays

JavaScript provides methods to sort arrays in different ways. The sort() method allows us to arrange elements alphabetically or numerically, while the reverse() method reverses the order of elements in an array.

Sorting Arrays with sort()

The sort() method sorts an array alphabetically by default.

let fruits = ["Banana", "Apple", "Cherry", "Date"];
fruits.sort();
console.log(fruits); // Output: ["Apple", "Banana", "Cherry", "Date"]

Sorting Numbers

By default, sort() treats numbers as strings, which can cause unexpected results. To sort numbers correctly, we need to provide a comparison function.

let numbers = [40, 5, 25, 10];
numbers.sort((a, b) => a - b);
console.log(numbers); // Output: [5, 10, 25, 40]

Reversing an Array

The reverse() method reverses the order of elements in an array.

let letters = ["A", "B", "C", "D"];
letters.reverse();
console.log(letters); // Output: ["D", "C", "B", "A"]

What will be the output of the following code?

cities.sort();

console.log(cities); // Output: ["Berlin", "London", "New York", "Paris"]

let numbers = [100, 20, 3, 50];
numbers.sort();
console.log(numbers);
console.log(numbers); // Output: [100, 20, 3, 50] (Sorted as strings)

3. Sorting Numbers Correctly

What will be the output of the following code?

let numbers = [100, 20, 3, 50];
numbers.sort((a, b) => a - b);
console.log(numbers);
console.log(numbers); // Output: [3, 20, 50, 100]

4. Reversing an Array

What will be the output of this code?

let letters = ["X", "Y", "Z"];
letters.reverse();
console.log(letters);
console.log(letters); // Output: ["Z", "Y", "X"]
