JS Multidimensional Arrays

A multidimensional array is an array that contains other arrays as its elements. These arrays allow us to store data in a structured way, such as representing tables or grids.

Creating a Multidimensional Array

A two-dimensional array (2D array) is an array of arrays. We can define one by nesting arrays inside another array.

let matrix = [
[1, 2, 3],
[4, 5, 6],
[7, 8, 9]
];
console.log(matrix);

Accessing Elements in a Multidimensional Array

To access an element in a 2D array, we use two indices: one for the row and one for the column.

let numbers = [
[10, 20, 30],
[40, 50, 60]
];
console.log(numbers[0][1]); // Output: 20
console.log(numbers[1][2]); // Output: 60

Modifying Elements in a Multidimensional Array

We can modify elements in a multidimensional array by directly accessing them using their row and column indices.

let data = [
[1, 2],
[3, 4]
];
data[1][0] = 99;
console.log(data); // Output: [[1, 2], [99, 4]]

Iterating Through a Multidimensional Array

We can use nested loops to iterate over a multidimensional array.

let matrix = [
[1, 2, 3],
[4, 5, 6]
];
for (let i = 0; i < matrix.length; i++) {
for (let j = 0; j < matrix[i].length; j++) {
console.log(matrix[i][j]);
}
}
