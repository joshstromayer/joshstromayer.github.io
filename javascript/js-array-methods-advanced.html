JS Advanced Array Methods

JavaScript provides powerful built-in methods to work with arrays more efficiently. These methods allow us to transform, filter, and extract data from arrays in a more readable way.

Common Advanced Array Methods

Here are some commonly used advanced array methods:

  • map() – Creates a new array by applying a function to each element.
  • filter() – Creates a new array containing only elements that meet a condition.
  • reduce() – Reduces an array to a single value by applying a function.

Using map()

The map() method is used to create a new array where each element is transformed based on a provided function.

let numbers = [1, 2, 3, 4];
let doubled = numbers.map(num => num * 2);
console.log(doubled); // Output: [2, 4, 6, 8]

Using filter()

The filter() method is used to create a new array containing only elements that satisfy a condition.

let numbers = [10, 15, 20, 25, 30];
let greaterThan20 = numbers.filter(num => num > 20);
console.log(greaterThan20); // Output: [25, 30]

Using reduce()

The reduce() method is used to reduce an array to a single value by applying a function.

let numbers = [1, 2, 3, 4];
let sum = numbers.reduce((total, num) => total + num, 0);
console.log(sum); // Output: 10

JS Advanced Array Methods: Exercises

1. Using map()

What will be the output of the following code?

let numbers = [2, 4, 6];
let squared = numbers.map(num => num ** 2);
console.log(squared);
console.log(squared); // Output: [4, 16, 36]

2. Using filter()

What will be the output of the following code?

let words = ["apple", "banana", "cherry", "date"];
let longWords = words.filter(word => word.length > 5);
console.log(longWords);
console.log(longWords); // Output: ["banana", "cherry"]

3. Using reduce()

What will be the output of the following code?

let numbers = [5, 10, 15];
let product = numbers.reduce((total, num) => total * num, 1);
console.log(product);
console.log(product); // Output: 750

4. Combining Methods

What will be the output of this code?

let numbers = [1, 2, 3, 4, 5, 6];
let result = numbers.filter(num => num % 2 === 0).map(num => num * 10);
console.log(result);
console.log(result); // Output: [20, 40, 60]
