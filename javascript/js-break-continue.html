Break and Continue

In JavaScript, the break and continue statements are used to control loop execution. The break statement is used to exit a loop entirely, while the continue statement skips the current iteration and moves to the next one.

The break Statement

The break statement immediately stops the loop execution and exits. This is useful when you want to end a loop early based on a condition.

Example: Stopping a loop when the number reaches 3

for (let i = 1; i <= 5; i++) {
    if (i === 3) break;
    console.log(i);
}

The loop starts at 1 and stops completely when i equals 3.

The continue Statement

The continue statement skips the current iteration and moves to the next loop cycle.

Example: Skipping even numbers

for (let i = 1; i <= 5; i++) {
    if (i % 2 === 0) continue;
    console.log(i);
}

In this example, when i is even, continue skips that iteration, so only odd numbers are printed.

Using break in a While Loop

The break statement can also be used in a while loop to exit early.

Example: Stopping a while loop when a number reaches 4

let num = 1;
while (num <= 5) {
    if (num === 4) break;
    console.log(num);
    num++;
}

Using continue in a While Loop

The continue statement skips a specific iteration in a while loop.

Example: Skipping multiples of 3

let x = 0;
while (x < 10) {
    x++;
    if (x % 3 === 0) continue;
    console.log(x);
}

When to Use Break and Continue?

- Use break when you need to exit a loop early.
- Use continue when you want to skip specific iterations.

1. Using break

Write a for loop that prints numbers from 1 to 10, but stops when it reaches 6.

for (let i = 1; i <= 10; i++) {

    console.log(i);
}

2. Using continue

Write a loop that prints numbers from 1 to 10, but skips numbers divisible by 4.

for (let i = 1; i <= 10; i++) {
    if (i % 4 === 0) continue;
    console.log(i);
}

    i++;

while (i <= 10) {
    if (i === 7) break;
