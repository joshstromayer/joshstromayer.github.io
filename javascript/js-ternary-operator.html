Ternary Operator

The ternary operator is a shorthand for writing simple if-else conditions in JavaScript. It allows you to evaluate a condition and return one of two values based on whether the condition is true or false.

Basic Syntax

The ternary operator uses the following syntax:

condition ? expression_if_true : expression_if_false;

If the condition evaluates to true, the first expression executes. Otherwise, the second expression runs.

Example: Checking Age

The example below uses a ternary operator to determine if a person is old enough to vote.

let age = 20;
let canVote = (age >= 18) ? "You can vote" : "You cannot vote";
console.log(canVote);

Replacing If-Else Statements

The ternary operator is useful for replacing simple if-else statements, making code more concise.

Example: Using if-else

let num = 5;
let result;
if (num > 0) {
    result = "Positive";
} else {
    result = "Negative";
}
console.log(result);

Example: Using the Ternary Operator

let num = 5;
let result = (num > 0) ? "Positive" : "Negative";
console.log(result);

Nesting Ternary Operators

You can nest ternary operators, but it can reduce readability if overused.

Example:

let score = 85;
let grade = (score >= 90) ? "A" :
    (score >= 80) ? "B" :
    (score >= 70) ? "C" : "F";
console.log("Grade:", grade);

When to Use the Ternary Operator?

- Use the ternary operator for short, simple conditions.
- Avoid using it for complex conditions, as it may reduce code readability.

Ternary Operator: Exercises

1. Basic Ternary Condition

Write a ternary statement that checks if a number is even or odd.

let num = 7;
let result = (num % 2 === 0) ? "Even" : "Odd";
console.log(result);

2. Voting Eligibility

Write a ternary statement that assigns "Eligible" if the age is 18 or older and "Not eligible" otherwise.

let age = 16;
let eligibility = (age >= 18) ? "Eligible" : "Not eligible";
console.log(eligibility);

3. Using a Nested Ternary

Use a nested ternary operator to categorize a number as "Positive", "Negative", or "Zero".

let number = 0;
let result = (number > 0) ? "Positive" :
    (number < 0) ? "Negative" : "Zero";
console.log(result);
