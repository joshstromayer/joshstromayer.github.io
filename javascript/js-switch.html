Switch Statements

In JavaScript, the switch statement is an alternative to using multiple if else conditions. It allows you to compare a value against multiple cases and execute the matching block of code.

Basic Syntax

A switch statement evaluates an expression and compares it to multiple case values. When a match is found, the corresponding block of code runs.

Example:

let day = "Monday";
switch (day) {
    case "Monday":
        console.log("Start of the work week.");
        break;
    case "Friday":
        console.log("Weekend is near!");
        break;
    default:
        console.log("It's a regular day.");
}

The break Keyword

The break statement is used to stop execution once a matching case is found. Without break, JavaScript will continue executing all subsequent cases.

Example Without break:

let color = "red";
switch (color) {
    case "red":
        console.log("Color is red");
    case "blue":
        console.log("Color is blue");
}

Output:

Color is red
Color is blue

Since there is no break, both cases execute.

The default Case

The default case runs when none of the cases match the given value.

Example:

let fruit = "Mango";
switch (fruit) {
    case "Apple":
        console.log("It's an apple.");
        break;
    default:
        console.log("Unknown fruit.");
}

When to Use a Switch Statement?

- Use switch when comparing a single variable against multiple values.
- Use if-else when conditions involve logical operators or complex conditions.

Switch Statements: Exercises

1. Basic Switch Case

Write a switch statement that logs "Good morning!" if the time is "morning", "Good afternoon!" if it's "afternoon", and "Good night!" for anything else.

let time = "morning";
switch (time) {
    case "morning":
        console.log("Good morning!");
        break;
    case "afternoon":
        console.log("Good afternoon!");
        break;
    default:
        console.log("Good night!");
}

2. Using break Correctly

Modify the following switch statement so only one message is logged:

let drink = "Tea";
switch (drink) {
    case "Tea":
        console.log("You chose Tea.");
    case "Coffee":
        console.log("You chose Coffee.");
}
let drink = "Tea";
switch (drink) {
    case "Tea":
        console.log("You chose Tea.");
        break;
    case "Coffee":
        console.log("You chose Coffee.");
}
