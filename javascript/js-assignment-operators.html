JS Assignment Operators

Assignment operators in JavaScript are used to assign values to variables. The most commonly used assignment operator is =, but JavaScript provides many other assignment operators that perform calculations while assigning values.

Basic Assignment Operator

The basic assignment operator = assigns a value to a variable.

let x = 10;
console.log(x); // Output: 10

Compound Assignment Operators

JavaScript also provides compound assignment operators that perform a mathematical operation and then assign the result to the variable.

  • += (Addition assignment) – Adds a value to a variable.
  • -= (Subtraction assignment) – Subtracts a value from a variable.
  • *= (Multiplication assignment) – Multiplies a variable by a value.
  • /= (Division assignment) – Divides a variable by a value.
  • %= (Modulus assignment) – Assigns the remainder after division.
  • **= (Exponentiation assignment) – Raises a variable to the power of a value.

Example: Compound Assignment Operators

let a = 5;
a += 3; // Same as: a = a + 3
console.log(a); // Output: 8

let b = 10;
b -= 4; // Same as: b = b - 4
console.log(b); // Output: 6

let c = 2;
c **= 3; // Same as: c = c ** 3
console.log(c); // Output: 8

Declare a variable num and assign it the value 20. Print its value using console.log().

Modify the following variable using assignment operators and predict the output.

let value = 50;
value += 10;
value -= 5;
console.log(value);
console.log(value); // Output: 55

3. Exponentiation Assignment

What will be the output of the following?

let num = 2;
num **= 4;
