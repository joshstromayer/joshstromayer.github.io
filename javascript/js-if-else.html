If, Else, and Else If

In JavaScript, conditional statements allow you to control the flow of your program based on different conditions. The if, else, and else if statements help execute different blocks of code depending on whether a condition is true or false.

The if Statement

The if statement runs a block of code only if a specified condition evaluates to true.

Example:

if (10 > 5) {
    console.log("10 is greater than 5");
}

The else Statement

The else statement provides an alternative block of code that runs if the if condition evaluates to false.

Example:

let age = 16;
if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}

The else if Statement

When you have multiple conditions to check, you can use the else if statement to test additional cases before falling back to else.

Example:

let score = 75;
if (score >= 90) {
    console.log("You got an A!");
} else if (score >= 80) {
    console.log("You got a B!");
} else if (score >= 70) {
    console.log("You got a C!");
} else {
    console.log("You need to study more!");
}

When to Use If, Else, and Else If?

- Use if when you need to check a single condition.
- Use else to handle cases when the condition is false.
- Use else if when you have multiple conditions to test.

If, Else, and Else If: Exercises

1. Basic If Statement

Write an if statement that checks if a number is greater than zero and logs "Positive number" if true.

let num = 5;
if (num > 0) {
    console.log("Positive number");
}

2. If-Else Condition

Write an if-else statement that checks if a user is old enough to vote (18 or older). If true, log "You can vote", otherwise log "You cannot vote".

let age = 17;
if (age >= 18) {
    console.log("You can vote");
} else {
    console.log("You cannot vote");
}

3. Else If Statement

Write an else if statement that checks if a number is positive, negative, or zero.

let number = -3;
if (number > 0) {
    console.log("Positive number");
} else if (number < 0) {
    console.log("Negative number");
} else {
    console.log("Zero");
}

4. Grading System

Create a program that takes a student's score and assigns a grade using if, else if, and else statements.

let score = 85;
if (score >= 90) {
    console.log("Grade: A");
} else if (score >= 80) {
    console.log("Grade: B");
} else if (score >= 70) {
    console.log("Grade: C");
} else {
    console.log("Grade: F");
}
