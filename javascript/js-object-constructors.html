JS Object Constructors

JavaScript provides a way to create multiple objects with similar properties and methods using **constructor functions**. A constructor function acts as a blueprint for creating objects dynamically.

Creating a Constructor Function

A constructor function is a regular JavaScript function, but by convention, its name starts with an uppercase letter. We use the new keyword to create new instances of an object.

function Person(name, age, city) {
this.name = name;
this.age = age;
this.city = city;
}
let person1 = new Person("Jessica", 30, "New York");
let person2 = new Person("James", 25, "Los Angeles");
console.log(person1.name); // Output: "Jessica"
console.log(person2.city); // Output: "Los Angeles"

Adding Methods to a Constructor

We can also define methods inside a constructor function.

function Car(brand, model, year) {
this.brand = brand;
this.model = model;
this.year = year;
this.getInfo = function() {
return this.brand + " " + this.model + " (" + this.year + ")";
};
}
let myCar = new Car("Toyota", "Corolla", 2022);
console.log(myCar.getInfo()); // Output: "Toyota Corolla (2022)"

Using the instanceof Operator

We can check if an object was created using a constructor function with the instanceof operator.

let student = new Person("Emma", 22, "Chicago");
console.log(student instanceof Person); // Output: true

JS Object Constructors: Exercises

1. Creating an Object Using a Constructor

What will be the output of the following code?

function Animal(type, sound) {
this.type = type;
this.sound = sound;
}
let dog = new Animal("Dog", "Bark");
console.log(dog.sound);
console.log(dog.sound); // Output: "Bark"

2. Checking instanceof

What will be the output of the following code?

function Laptop(brand, ram) {
this.brand = brand;
this.ram = ram;
}
let myLaptop = new Laptop("Dell", "16GB");
console.log(myLaptop instanceof Laptop);
console.log(myLaptop instanceof Laptop); // Output: true

3. Adding a Method to a Constructor

What will be the output of the following code?

function Phone(brand, model) {
this.brand = brand;
this.model = model;
this.getDetails = function() {
return this.brand + " " + this.model;
};
}
let myPhone = new Phone("Apple", "iPhone 14");
console.log(myPhone.getDetails());
console.log(myPhone.getDetails()); // Output: "Apple iPhone 14"

4. Constructor vs. Object Literal

Which of the following is the correct way to create a new instance using a constructor function?

function User(username) {
this.username = username;
}
let user1 = new User("StevieDoe");
let user2 = User("JaneDoe");

The correct way is let user1 = new User("StevieDoe");. The second statement (let user2 = User("JaneDoe");) is incorrect because the new keyword is missing.
