JS Object Prototypes

In JavaScript, every object has a built-in property called **prototype**, which allows us to add new properties and methods to object constructors. This is useful for sharing methods across multiple object instances without duplicating them.

Understanding Prototypes

Every JavaScript object is linked to a **prototype object** from which it can inherit properties and methods. We can modify the prototype of a constructor function to add methods for all instances.

Adding Methods to a Constructor Using Prototypes

Instead of defining methods inside the constructor, we can use the prototype property to attach methods to all instances of an object.

function Person(name, age) {
this.name = name;
this.age = age;
}
Person.prototype.greet = function() {
return "Hello, my name is " + this.name;
};
let person1 = new Person("Jessica", 30);
console.log(person1.greet()); // Output: "Hello, my name is Jessica"

Adding Properties Using Prototypes

We can also add new properties to an objectâ€™s prototype, and all instances will inherit them.

function Car(brand, model) {
this.brand = brand;
this.model = model;
}
Car.prototype.year = 2022;
let myCar = new Car("Toyota", "Corolla");
console.log(myCar.year); // Output: 2022

Checking Prototype Properties

The hasOwnProperty() method allows us to check if a property belongs directly to an object or if it is inherited from the prototype.

console.log(myCar.hasOwnProperty("brand")); // Output: true
console.log(myCar.hasOwnProperty("year")); // Output: false (inherited from prototype)

JS Object Prototypes: Exercises

1. Adding a Method to a Prototype

What will be the output of the following code?

function Animal(type) {
this.type = type;
}
Animal.prototype.sound = function() {
return "Some sound";
};
let dog = new Animal("Dog");
console.log(dog.sound());
console.log(dog.sound()); // Output: "Some sound"

2. Inheriting Prototype Properties

What will be the output of the following code?

function Laptop(brand) {
this.brand = brand;
}
Laptop.prototype.memory = "16GB";
let myLaptop = new Laptop("HP");
console.log(myLaptop.memory);
console.log(myLaptop.memory); // Output: "16GB"

3. Checking Property Ownership

What will be the output of the following code?

function Phone(model) {
this.model = model;
}
Phone.prototype.brand = "Samsung";
let myPhone = new Phone("Galaxy S21");
console.log(myPhone.hasOwnProperty("brand"));
console.log(myPhone.hasOwnProperty("brand")); // Output: false (it's inherited)

4. Prototype vs. Own Properties

What will be the output of this code?

function Student(name) {
this.name = name;
}
Student.prototype.school = "High School";
let student1 = new Student("Liam");
student1.school = "University";
console.log(student1.school);
console.log(student1.school); // Output: "University" (overrides prototype property)
